# First API call in 3 minutes

Get up and running with the NAWA classification API in three steps.

<Steps>
  <Step title="Get your API key">
    Go to [trynawa.com/developers/keys](https://trynawa.com/developers/keys) to create your free API key. Sign in with Google or email -- no credit card required.

    <Note>
      Free keys (`nawa_test_sk_xxx`) require no credit card. Limited to 100 lifetime requests and 14-day expiry. They use real AI models -- same accuracy as paid keys.
    </Note>
  </Step>

  <Step title="Make your first classification">
    Send a comment to the `/v1/classify` endpoint:

    <CodeGroup>

    ```bash cURL
    curl -X POST https://api.trynawa.com/v1/classify \
      -H "Authorization: Bearer nawa_test_sk_xxx" \
      -H "Content-Type: application/json" \
      -d '{"text": "متى الجزء الثاني؟"}'
    ```

    ```typescript TypeScript
    const response = await fetch('https://api.trynawa.com/v1/classify', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer nawa_test_sk_xxx',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({ text: 'متى الجزء الثاني؟' })
    })

    const data = await response.json()
    console.log(data.result.intent)   // ["question"]
    console.log(data.result.dialect)  // "gulf"
    ```

    ```python Python
    import requests

    response = requests.post(
        "https://api.trynawa.com/v1/classify",
        headers={
            "Authorization": "Bearer nawa_test_sk_xxx",
            "Content-Type": "application/json"
        },
        json={"text": "متى الجزء الثاني؟"}
    )

    data = response.json()
    print(data["result"]["intent"])   # ["question"]
    print(data["result"]["dialect"])  # "gulf"
    ```

    </CodeGroup>
  </Step>

  <Step title="Read the response">
    Every NAWA response follows a consistent envelope format:

    ```json
    {
      "result": {
        "id": "cls_nw_a1b2c3d4e5f6",
        "object": "classification",
        "intent": ["question"],
        "sentiment": "neutral",
        "language": "ar",
        "dialect": "gulf",
        "dialect_confidence": 0.95,
        "requires_response": true,
        "priority": "high",
        "suggested_reply": {
          "text": "سؤال عن موعد الجزء التالي من المحتوى",
          "direction": "rtl"
        },
        "provider": "allam",
        "model": "sdaia/allam-1-13b-instruct",
        "fallback_used": false,
        "tokens_used": null,
        "cost_usd": 0.006,
        "credits_used": 6
      },
      "success": true,
      "errors": [],
      "request_id": "req_nw_a1b2c3d4e5f67890abcdef12"
    }
    ```

    **Key response fields:**

    | Field | Type | Description |
    |-------|------|-------------|
    | `intent` | string[] | Detected intents (e.g., `["question"]`, `["question", "complaint"]`) |
    | `sentiment` | string | `positive`, `negative`, `neutral`, or `mixed` |
    | `dialect` | string \| null | `gulf`, `egyptian`, `levantine`, `maghrebi`, `msa`, or null for non-Arabic |
    | `dialect_confidence` | number \| null | Confidence score (0-1) |
    | `requires_response` | boolean | Whether the comment warrants a reply |
    | `priority` | string | `high`, `medium`, or `low` |
    | `provider` | string | AI provider used: `claude`, `gemini`, or `allam` |
    | `cost_usd` | number | Cost in USD |
    | `credits_used` | integer | Credits deducted |
  </Step>
</Steps>
