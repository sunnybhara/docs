<Update label="March 18, 2026" description="v1.1.1" tags={["API", "Reliability"]}>
## Claude Haiku 4.5 fallback for translate and moderate

The `/v1/translate` and `/v1/moderate` endpoints now use an ALLaM-to-Claude fallback chain. If ALLaM is unavailable, requests are automatically routed to Claude Haiku 4.5 (`claude-haiku-4-5-20251001`).

### What changed
- Added `fallback_used` field to all endpoint responses for consistency
- When `fallback_used` is `true`, the `provider` field shows `claude` and `model` shows `claude-haiku-4-5-20251001`
- No changes to request parameters, pricing, or rate limits
- Fallback is transparent to callers and requires no code changes
</Update>

<Update label="March 18, 2026" description="v1.1.0" tags={["API", "New Endpoint"]}>
## New endpoints: Translate, Detect, and Moderate

Three new endpoints expanding NAWA from comment classification to a full Arabic language intelligence platform.

### POST /v1/translate
Dialect-aware Arabic translation with tone control.
- 5 credits per request ($0.005)

### POST /v1/detect
Lightweight language and dialect detection in under 100ms.
- 2 credits per request ($0.002)

### POST /v1/moderate
Dialect-aware Arabic content moderation. Understands profanity and slang differences across Gulf, Egyptian, Levantine, and MSA Arabic.
- 4 credits per request ($0.004)
- Categories: profanity, hate speech, harassment, sexual, spam, self-harm, violence
- Returns severity scores, flagged terms, and cleaned text
</Update>
