<Update label="March 18, 2026" description="v1.1.0" tags={["API", "New Endpoint"]}>
## New endpoints: Translate and Detect

Two new endpoints expanding NAWA from comment classification to a full Arabic language intelligence platform.

### POST /v1/translate
Dialect-aware Arabic translation with tone control.
- 5 credits per request ($0.005)
- Supports: formal, casual, social_media tones
- Dialect-to-dialect conversion (e.g., MSA to Gulf)

### POST /v1/detect
Lightweight language and dialect detection in under 100ms. No AI model call required.
- 2 credits per request ($0.002)
- Detects: language, dialect, confidence, script, text direction, word count
- Ideal as a routing layer before classify or translate
</Update>

## NAWA API Platform v1.0 -- Live

Self-serve API platform for Arabic comment classification.

### What shipped
- 8 API endpoints at api.trynawa.com/v1
- Developer console at trynawa.com/developers/keys
- Google OAuth + magic link authentication
- Credit-based billing via Stripe ($25/$50/$100/$500 packs)
- Free tier: 100 requests, 14-day expiry
- Live tier: no cap, no expiry
- TypeScript SDK (@nawalabs/sdk) and Python SDK (nawa-sdk)
- Interactive docs at developers.trynawa.com
- OpenAPI 3.1 spec
- RLHF calibration loop (pattern applicator + calibration endpoints)
- Gulf dialect detection verified in production via HUMAIN ALLaM 13B
</Update>
