Classify a comment and generate an AI-powered reply in one call. The reply matches the commenter's dialect and respects the specified tone and length constraints.

<Note>
  Cost: **$0.008** per request (8 credits).
</Note>

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify and reply to. |
| `tone` | string | No | Reply tone: `friendly`, `professional`, `casual`, `formal`. Default: `professional`. |
| `max_length` | integer | No | Maximum reply length in characters (1-2000). Default: `500`. |
| `context` | object | No | Optional context for reply generation. |
| `context.platform` | string | No | Source platform for platform-specific context. |
| `context.brand_voice` | string | No | Brand voice guidelines to follow. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/comments/reply \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "هذا المنتج سيء جداً ولا أنصح به",
    "tone": "professional",
    "max_length": 500,
    "context": {
      "platform": "youtube",
      "brand_voice": "Empathetic and solution-oriented"
    }
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.comments.reply({
  text: 'هذا المنتج سيء جداً ولا أنصح به',
  tone: 'professional',
  maxLength: 500,
  context: {
    platform: 'youtube',
    brandVoice: 'Empathetic and solution-oriented'
  }
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.comments.reply(
    text="هذا المنتج سيء جداً ولا أنصح به",
    tone="professional",
    max_length=500,
    context={
        "platform": "youtube",
        "brand_voice": "Empathetic and solution-oriented",
    },
)
```

</CodeGroup>

```json
{
  "result": {
    "id": "rpl_nw_a1b2c3d4e5f6",
    "object": "comment_reply",
    "classification": {
      "intent": ["complaint"],

      "priority": "high",
      "requires_response": true

    "reply": {
      "text": "نعتذر عن تجربتك السلبية. نود أن نفهم المشكلة بشكل أفضل لنتمكن من مساعدتك. يرجى التواصل معنا عبر الرسائل الخاصة.",
      "direction": "rtl",

    "provider": "claude",
    "model": "claude-3-5-sonnet-20241022",

  },
  "success": true,
  "errors": [],
  "request_id": "req_nw_a1b2c3d4e5f67890abcdef12"

```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Reply ID (`rpl_nw_` prefix) |
| `object` | string | Always `comment_reply` |
| `classification.intent` | string[] | Detected intents of the original comment |
| `classification.sentiment` | string | Sentiment of the original comment |
| `classification.priority` | string | Priority level: `high`, `medium`, `low` |
| `classification.requires_response` | boolean | Whether the comment warrants a reply |
| `reply.text` | string | The generated reply text |
| `reply.direction` | string | Text direction: `rtl` or `ltr` |
| `reply.tone` | string | The tone used for the reply |
| `provider` | string | AI provider used |
| `model` | string | Model version used |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits consumed |
