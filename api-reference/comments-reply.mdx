Classify a comment and generate an AI-powered reply in a single request. The reply matches the commenter's dialect and cultural context.

<Note>
  Cost: **$0.008** per request (8 credits).
</Note>

## Request

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify and reply to. |
| `tone` | string | No | Reply tone: `friendly`, `professional`, `casual`, `formal`. Default: `professional`. |
| `max_length` | integer | No | Maximum reply length in characters (1-2000). Default: 500. |
| `context` | object | No | Additional context. Supports `platform` (string) and `brand_voice` (string) fields. |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, `allam`. Default: auto-routed. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/comments/reply \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟",
    "tone": "friendly",
    "context": {
      "platform": "youtube",
      "brand_voice": "Tech review channel focused on smartphones"
    }
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.comments.reply({
  text: 'متى الجزء الثاني؟',
  tone: 'friendly',
  context: {
    platform: 'youtube',
    brandVoice: 'Tech review channel focused on smartphones'
  }
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.comments.reply(
    text="متى الجزء الثاني؟",
    tone="friendly",
    context={
        "platform": "youtube",
        "brand_voice": "Tech review channel focused on smartphones"
    }
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique reply ID (`rpl_nw_` prefix) |
| `object` | string | Always `"comment_reply"` |
| `classification.intent` | string[] | Detected intents of the original comment |
| `classification.sentiment` | string | Detected sentiment: `positive`, `negative`, `neutral`, `mixed` |
| `classification.priority` | string | Priority level for response |
| `classification.requires_response` | boolean | Whether the comment warrants a reply |
| `reply.text` | string | The generated reply text |
| `reply.direction` | string | Text direction: `rtl` for Arabic, `ltr` for English |
| `reply.tone` | string | The tone used for the reply |
| `provider` | string | AI provider used |
| `model` | string | Specific model identifier |
| `cost_usd` | number | Cost of this request in USD |
| `credits_used` | integer | Credits deducted |

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier |
| `X-NAWA-Provider` | AI provider used |
| `X-NAWA-Latency` | Request latency in milliseconds |
