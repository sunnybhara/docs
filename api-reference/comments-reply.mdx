Classify a comment and generate a contextual reply in a single call. The reply matches the commenter's language direction and respects your tone and length settings.

<Note>
  Cost: **$0.008** per request (8 credits).
</Note>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, or `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify and reply to. Must be non-empty. |
| `tone` | string | No | Reply tone: `friendly`, `professional`, `casual`, or `formal`. Default: `professional`. |
| `max_length` | integer | No | Maximum reply length in characters (1-2000). Default: `500`. |
| `context` | object | No | Additional context. Supports `platform` (string) and `brand_voice` (string). |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/comments/reply \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "هذا المنتج سيء جداً ولا أنصح به",
    "tone": "professional",
    "max_length": 500
  }'
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/comments/reply', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer nawa_test_sk_xxx',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'هذا المنتج سيء جداً ولا أنصح به',
    tone: 'professional',
    max_length: 500
  })
})

const data = await response.json()
console.log(data.result.reply.text)       // generated reply
console.log(data.result.classification)   // classification details
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/comments/reply",
    headers={
        "Authorization": "Bearer nawa_test_sk_xxx",
        "Content-Type": "application/json"
    },
    json={
        "text": "هذا المنتج سيء جداً ولا أنصح به",
        "tone": "professional",
        "max_length": 500
    }
)

data = response.json()
print(data["result"]["reply"]["text"])       # generated reply
print(data["result"]["classification"])      # classification details
```

</CodeGroup>


### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Reply ID (`rpl_nw_xxx` format) |
| `object` | string | Always `comment_reply` |
| `classification.intent` | string[] | Detected intents from the original comment |
| `classification.sentiment` | string | `positive`, `negative`, `neutral`, or `mixed` |
| `classification.priority` | string | `high`, `medium`, or `low` |
| `classification.requires_response` | boolean | Whether the comment warrants a reply |
| `reply.text` | string | The generated reply text |
| `reply.direction` | string | Text direction: `rtl` for Arabic, `ltr` for English |
| `reply.tone` | string | The tone used for the reply |
| `provider` | string | AI provider used |
| `model` | string | Model version used |
| `cost_usd` | number | Cost in USD for this request |
| `credits_used` | integer | Credits deducted from your balance |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, invalid `tone`, `max_length` out of range, invalid `provider` |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `permission_error` | Insufficient credits |
| 405 | `invalid_request_error` | Non-POST request method |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
