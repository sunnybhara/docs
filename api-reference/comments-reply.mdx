Classify a comment and generate a culturally-aware reply in one request. The reply matches the commenter's dialect and respects the configured tone.

<Note>
  Cost: **$0.008** per request (8 credits).
</Note>

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify and reply to. Max 5,000 characters. |
| `tone` | string | No | Reply tone: `friendly`, `professional`, `casual`, `formal`. Default: `professional`. |
| `max_length` | integer | No | Maximum reply length in characters (1-2000). Default: `500`. |
| `context` | object | No | Optional context object with `platform` (string) and `brand_voice` (string) fields. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/comments/reply \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "هذا المنتج سيء جدا ولا أنصح به",
    "tone": "professional",
    "max_length": 500,
    "context": {
      "platform": "youtube",
      "brand_voice": "Empathetic and solution-oriented"
    }
  }'
```

```typescript TypeScript
const { data, error } = await nawa.comments.reply({
  text: 'هذا المنتج سيء جدا ولا أنصح به',
  tone: 'professional',
  maxLength: 500,
  context: {
    platform: 'youtube',
    brandVoice: 'Empathetic and solution-oriented'
  }
})
```

```python Python
result = nawa.comments.reply(
    text="هذا المنتج سيء جدا ولا أنصح به",
    tone="professional",
    max_length=500,
    context={
        "platform": "youtube",
        "brand_voice": "Empathetic and solution-oriented",
    },
)
```

</CodeGroup>

```json
{
  "success": true,
  "result": {

    "object": "comment_reply",
    "classification": {

      "direction": "rtl",

    "cost_usd": 0.008,

```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Reply ID (`rpl_nw_xxx` format) |
| `object` | string | Always `comment_reply` |
| `classification` | object | Inline classification with `intent` (string[]), `sentiment`, `priority`, `requires_response` |
| `reply` | object | Generated reply with `text`, `direction` (ltr/rtl), and `tone` |
| `provider` | string | AI provider used |
| `model` | string | Model version used |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits deducted |
