Classify a comment and generate a contextual reply in a single call. The reply matches the commenter's dialect and is contextually appropriate. Supports tone control and max length configuration.

<Note>
  Cost: **$0.008** per request (8 credits). This is the most expensive endpoint because it performs classification and reply generation in one call.
</Note>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `text` | string | Yes | -- | The comment text to classify and reply to. |
| `tone` | string | No | `professional` | Reply tone: `friendly`, `professional`, `casual`, `formal`. |
| `max_length` | integer | No | `500` | Maximum reply length in characters (1-2000). |
| `context` | object | No | -- | Additional context with optional `platform` and `brand_voice` fields. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/comments/reply \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "هذا المنتج سيء جداً ولا أنصح به",
    "tone": "professional",
    "max_length": 500
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const result = await nawa.comments.reply({
  text: 'هذا المنتج سيء جداً ولا أنصح به',
  tone: 'professional',
  maxLength: 500
})

console.log(result.data.classification.intent)  // ["complaint"]
console.log(result.data.reply.text)             // Generated reply
console.log(result.data.reply.direction)        // "rtl"
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.comments.reply(
    text="هذا المنتج سيء جداً ولا أنصح به",
    tone="professional",
    max_length=500
)

print(result.data.classification.intent)  # ["complaint"]
print(result.data.reply.text)             # Generated reply
print(result.data.reply.direction)        # "rtl"
```

</CodeGroup>

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "id": "rpl_nw_a1b2c3d4e5f6",
    "object": "comment_reply",
    "classification": {
      "intent": ["complaint"],
      "sentiment": "negative",
      "priority": "high",
      "requires_response": true
    },
    "reply": {
      "text": "نعتذر عن تجربتك السيئة. نحرص على جودة منتجاتنا ونود مساعدتك. تواصل معنا للدعم.",
      "direction": "rtl",
      "tone": "professional"
    },
    "provider": "allam",
    "model": "sdaia/allam-1-13b-instruct",
    "cost_usd": 0.008,
    "credits_used": 8
  },
  "errors": [],
  "request_id": "req_nw_rpl1a2b3c4d5e"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Reply ID in `rpl_nw_xxx` format |
| `object` | string | Always `"comment_reply"` |
| `classification` | object | Classification of the original comment |
| `classification.intent` | string[] | Detected intents of the original comment |
| `classification.sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `classification.priority` | string | `high`, `medium`, `low` |
| `classification.requires_response` | boolean | Whether this comment warrants a response |
| `reply` | object | The generated reply |
| `reply.text` | string | The reply text |
| `reply.direction` | string | Text direction: `rtl` or `ltr` |
| `reply.tone` | string | The tone used: `friendly`, `professional`, `casual`, `formal` |
| `provider` | string | AI provider used |
| `model` | string | Specific model version used |
| `cost_usd` | number | Cost in USD (always 0.008) |
| `credits_used` | integer | Credits deducted (always 8) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing or empty `text`. Invalid tone. `max_length` out of range. |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
