Translate text between English and Arabic with dialect awareness. Generates culturally appropriate translations in Gulf, Egyptian, Levantine, or MSA Arabic. Uses ALLaM as the primary provider with Claude Haiku 4.5 as a fallback.

<Note>
  Cost: **$0.005** per request (5 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to translate. Max 5,000 characters. |
| `source` | string | No | Source language: `ar`, `en`, `auto`. Default: `auto` (detects automatically). |
| `target` | string | Yes | Target language: `ar`, `en`. Must be different from source. |
| `dialect` | string | No | Target dialect for Arabic output: `gulf`, `egyptian`, `levantine`, `msa`. Default: `msa`. When translating from Arabic with no dialect specified, the source dialect is auto-detected. |
| `tone` | string | No | Translation tone: `formal`, `casual`, `social_media`. Default: `formal`. |
| `platform` | string | No | Source platform for tone context: `youtube`, `instagram`, `twitter`, `facebook`. |

### Example request

<CodeGroup>
```bash cURL
curl -X POST https://api.trynawa.com/v1/translate \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello, how are you?",
    "source": "en",
    "target": "ar",
    "dialect": "gulf",
    "tone": "casual"
  }'
```
```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.translate({
  text: 'Hello, how are you?',
  source: 'en',
  target: 'ar',
  dialect: 'gulf',
  tone: 'casual'
})
```
```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.translate(
    text="Hello, how are you?",
    source="en",
    target="ar",
    dialect="gulf",
    tone="casual"
)
```
</CodeGroup>

## Response

### Success response (200)
```json
{
  "success": true,
  "result": {
    "id": "trn_nw_pon1yom9bkfe",
    "object": "translation",
    "text": "Hello, how are you?",
    "translated_text": "هلا، كيف حالك؟",
    "source_language": "en",
    "target_language": "ar",
    "source_dialect": null,
    "target_dialect": "gulf",
    "tone": "casual",
    "provider": "allam",
    "model": "sdaia/allam-1-13b-instruct",
    "fallback_used": false,
    "tokens_used": null,
    "cost_usd": 0.005,
    "credits_used": 5,
    "cached": false
  },
  "errors": [],
  "request_id": "req_nw_4c11w71e44cb"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Translation ID (`trn_nw_` prefix) |
| `object` | string | Always `translation` |
| `text` | string | The original input text |
| `translated_text` | string | The translated output |
| `source_language` | string | Detected or specified source language (`ar` or `en`) |
| `target_language` | string | Target language |
| `source_dialect` | string or null | Detected dialect if source is Arabic |
| `target_dialect` | string or null | Target dialect if target is Arabic |
| `tone` | string | The tone used for translation |
| `provider` | string | AI provider used (`allam` or `claude`) |
| `model` | string | Specific model identifier used |
| `fallback_used` | boolean | `true` if ALLaM failed and Claude was used instead |
| `tokens_used` | integer or null | Token count (null when not tracked) |
| `cost_usd` | number | Cost in USD |
| `credits_used` | number | Credits deducted |
| `cached` | boolean | Whether served from semantic cache |

### Response headers

| Header | Description |
|--------|-------------|
| `X-NAWA-Cache` | `HIT` if served from cache (no cost), `MISS` otherwise |
| `X-NAWA-Provider` | AI provider used (`allam` or `claude`) |
| `X-NAWA-Latency` | Server-side processing time in milliseconds |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or `target`. Invalid values. Source equals target. Text too long (over 5,000 characters). |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
