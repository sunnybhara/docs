Moderate Arabic or English text for harmful content. NAWA understands profanity and slang differences across Arabic dialects, so region-specific offensive language is correctly flagged regardless of dialect.

<Note>
  Cost: **$0.004** per request (4 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`). Cache TTL is 300 seconds.
</Note>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to moderate. Max 5,000 characters. |
| `strictness` | string | No | Moderation strictness: `low`, `medium`, `high`. Default: `medium`. |
| `context` | string | No | Additional context to improve accuracy (e.g. "gaming community forum"). |
| `platform` | string | No | Source platform: `youtube`, `instagram`, `twitter`, `facebook`. |

### Strictness levels

| Level | Behavior |
|-------|----------|
| `low` | Flags only severe content: explicit hate speech, extreme profanity, direct threats. |
| `medium` | Flags moderate and severe content: profanity, harassment, hate speech, spam. |
| `high` | Flags all potentially problematic content: mild profanity, borderline content, subtle insults. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/moderate \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "هذا المحتوى رائع ومفيد جداً",
    "strictness": "medium",
    "platform": "youtube"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/moderate",
    headers={
        "Authorization": "Bearer nawa_test_sk_xxx",
        "Content-Type": "application/json",
    },
    json={
        "text": "هذا المحتوى رائع ومفيد جداً",
        "strictness": "medium",
        "platform": "youtube",
    },
)

result = response.json()
print(result["result"]["verdict"])  # "safe" or "flagged"
```

```typescript TypeScript
const response = await fetch("https://api.trynawa.com/v1/moderate", {
  method: "POST",
  headers: {
    Authorization: "Bearer nawa_test_sk_xxx",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    text: "هذا المحتوى رائع ومفيد جداً",
    strictness: "medium",
    platform: "youtube",
  }),
});

const { result } = await response.json();
console.log(result.verdict); // "safe" or "flagged"
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost) |
| `X-NAWA-Provider` | AI provider used (`allam`) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Moderation ID (`mod_nw_xxx` format) |
| `object` | string | Always `moderation` |
| `text` | string | The original input text |
| `is_safe` | boolean | Whether the content passed moderation |
| `verdict` | string | `safe` or `flagged` |
| `flags` | string[] | List of triggered category names |
| `severity` | string | `none`, `low`, `medium`, `high` |
| `severity_score` | number | Overall severity score (0.0 to 1.0) |
| `language` | string | Detected language: `ar`, `en`, `mixed` |
| `dialect` | string or null | Detected Arabic dialect: `gulf`, `egyptian`, `levantine`, `msa`. Null if not Arabic. |
| `categories` | object | Boolean flags for each moderation category |
| `category_scores` | object | Confidence scores (0.0 to 1.0) for each category |
| `flagged_terms` | string[] | Specific problematic words or phrases detected |
| `cleaned_text` | string or null | Text with flagged terms replaced by `***`. Null if content is safe. |
| `model` | string | Model version used |
| `provider` | string | AI provider used |
| `cached` | boolean | Whether served from semantic cache |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits deducted |

### Moderation categories

| Category | Description |
|----------|-------------|
| `profanity` | Profane language, swear words, vulgar expressions |
| `hate_speech` | Content targeting groups based on identity |
| `harassment` | Bullying, intimidation, personal attacks |
| `sexual` | Sexually explicit or suggestive content |
| `spam` | Unsolicited promotional or repetitive content |
| `self_harm` | Content promoting or glorifying self-harm |
| `violence` | Content promoting or glorifying violence |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, invalid `strictness`, text too long |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |

<Tip>
  NAWA uses a fail-open policy for moderation. If the AI model response cannot be parsed, the content defaults to `is_safe: true` so legitimate content is never blocked due to a processing error.
</Tip>
