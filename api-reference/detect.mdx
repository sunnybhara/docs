title: "Detect Language"
sidebarTitle: "POST /v1/detect"
description: "Lightweight language and dialect detection for Arabic and English text."
api: "POST https://api.trynawa.com/v1/detect"
---

Detect the language, dialect, and script of Arabic or English text. This is a lightweight, local-only endpoint with no external AI call, targeting sub-100ms response times.

## Request

<Note>
  Cost: **$0.002** per request (2 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to analyze. Max 5,000 characters. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/detect \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "كيف حالك يا صديقي؟"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.detect({
  text: 'كيف حالك يا صديقي؟'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.detect(
    text="كيف حالك يا صديقي؟"
)
```

</CodeGroup>

## Response

```json
{
  "success": true,
  "result": {
    "id": "det_nw_a1b2c3d4e5f6",
    "object": "detection",
    "text": "كيف حالك يا صديقي؟",
    "language": "ar",
    "dialect": "msa",
    "dialect_confidence": 0.88,
    "script": "arabic",
    "text_direction": "rtl",
    "word_count": 4,
    "cost_usd": 0.002,
    "credits_used": 2,
    "cached": false
  },
  "errors": [],
  "request_id": "req_nw_a1b2c3d4e5f67890abcdef12"
}
```

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Detection ID (`det_nw_xxx` format) |
| `object` | string | Always `detection` |
| `text` | string | The original input text |
| `language` | string | Detected language: `ar`, `en`, or `mixed` |
| `dialect` | string \| null | Arabic dialect if detected: `gulf`, `egyptian`, `levantine`, `maghrebi`, `msa` |
| `dialect_confidence` | number \| null | Confidence score for dialect detection (0-1) |
| `script` | string | Script type: `arabic`, `latin`, or `mixed` |
| `text_direction` | string | Text direction: `rtl` or `ltr` |
| `word_count` | integer | Number of words in the input |
| `cost_usd` | number | Cost in USD for this request |
| `credits_used` | integer | Credits deducted from your balance |
| `cached` | boolean | Whether this was served from semantic cache |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or text exceeds 5,000 characters |
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
