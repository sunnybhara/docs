Check the operational status of the NAWA API. This endpoint is **free** and does **not** require authentication.

<Note>
  The health endpoint returns a plain JSON object, not the standard success envelope used by other endpoints.
</Note>

## Request

<CodeGroup>

```bash cURL
curl https://api.trynawa.com/v1/health
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/health')
const data = await response.json()
console.log(data.status)  // "healthy" or "degraded"
```

```python Python
import requests

response = requests.get("https://api.trynawa.com/v1/health")
data = response.json()
print(data["status"])  # "healthy" or "degraded"
```

</CodeGroup>

## Response

### Healthy response (200)

```json
{
  "status": "healthy",
  "version": "v1",
  "services": {
    "database": { "status": "healthy", "latency_ms": 12 },
    "redis": { "status": "healthy", "latency_ms": 5 },
    "nagl": { "status": "healthy" }
  },
  "timestamp": "2025-01-15T12:00:00.000Z",
  "latency_ms": 15
}
```

### Degraded response (503)

```json
{
  "status": "degraded",
  "version": "v1",
  "services": {
    "database": { "status": "healthy", "latency_ms": 12 },
    "redis": { "status": "down", "latency_ms": 0 },
    "nagl": { "status": "healthy" }
  },
  "timestamp": "2025-01-15T12:00:00.000Z",
  "latency_ms": 15
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Overall status: `healthy` or `degraded` |
| `version` | string | API version (always `v1`) |
| `services.database` | object | Database status with `status` and `latency_ms` |
| `services.redis` | object | Redis cache status with `status` and `latency_ms` |
| `services.nagl` | object | NAGL classification engine status |
| `timestamp` | string | ISO 8601 timestamp of the check |
| `latency_ms` | integer | Total health check duration in milliseconds |

Each service `status` is one of: `healthy`, `degraded`, or `down`.

<Note>
  For real-time status monitoring, visit [status.trynawa.com](https://status.trynawa.com).
</Note>
