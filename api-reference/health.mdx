Check the operational status of the NAWA API and its dependencies. This endpoint is **free** and does **not** require authentication.

<Warning>
  This endpoint returns a bare JSON object, not the standard `{success, result, errors, request_id}` envelope used by other endpoints.
</Warning>

## Request

<CodeGroup>

```bash cURL
curl https://api.trynawa.com/v1/health
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/health')
const health = await response.json()
```

```python Python
import requests

response = requests.get("https://api.trynawa.com/v1/health")
health = response.json()
```

</CodeGroup>

## Response

### Healthy response (200)

```json
{
  "status": "healthy",
  "version": "v1",
  "services": {
    "database": {
      "status": "healthy",
      "latency_ms": 12
    },
    "redis": {
      "status": "healthy",
      "latency_ms": 5
    },
    "nagl": {
      "status": "healthy"
    }
  },
  "timestamp": "2025-01-15T12:00:00.000Z",
  "latency_ms": 25
}
```

### Degraded response (503)

```json
{
  "status": "degraded",
  "version": "v1",
  "services": {
    "database": {
      "status": "degraded",
      "latency_ms": 850
    },
    "redis": {
      "status": "healthy",
      "latency_ms": 5
    },
    "nagl": {
      "status": "healthy"
    }
  },
  "timestamp": "2025-01-15T12:00:00.000Z",
  "latency_ms": 860
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Overall status: `healthy` or `degraded` |
| `version` | string | API version (always `v1`) |
| `services.database` | object | Database status with `status` and `latency_ms` |
| `services.redis` | object | Redis cache status with `status` and `latency_ms` |
| `services.nagl` | object | NAGL engine status with `status` only |
| `timestamp` | string | ISO 8601 timestamp |
| `latency_ms` | integer | Total health check latency |

<Note>
  Returns HTTP 200 when all services are healthy, HTTP 503 when any service is degraded or down. For real-time status monitoring, visit [status.trynawa.com](https://status.trynawa.com).
</Note>
