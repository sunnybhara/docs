Check the operational status of the NAWA API. This endpoint is **free** and does **not** require authentication.

## Request

<CodeGroup>

```bash cURL
curl https://api.trynawa.com/v1/health
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/health')
const data = await response.json()
console.log(data.result.status) // "healthy" or "degraded"
```

```python Python
import requests

response = requests.get("https://api.trynawa.com/v1/health")
data = response.json()
print(data["result"]["status"])  # "healthy" or "degraded"
```

</CodeGroup>

## Response

### Healthy response (200)

```json
{
  "result": {
    "status": "healthy",
    "version": "1.0.0",
    "services": {
      "database": "operational",
      "redis": "operational",
      "nagl": "operational"
    },
    "timestamp": "2026-01-15T12:00:00Z",
    "latency_ms": 45
  },
  "success": true,
  "errors": [],
  "request_id": "req_nw_hlt_abc123"
}
```

### Degraded response (503)

When one or more services are not fully operational, the status is `degraded` and the HTTP status code is `503`:

```json
{
  "result": {
    "status": "degraded",
    "version": "1.0.0",
    "services": {
      "database": "operational",
      "redis": "degraded",
      "nagl": "operational"
    },
    "timestamp": "2026-01-15T12:00:00Z",
    "latency_ms": 1200
  },
  "success": true,
  "errors": [],
  "request_id": "req_nw_hlt_def456"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Overall system status: `healthy` or `degraded` |
| `version` | string | Current API version |
| `services.database` | string | Supabase database status |
| `services.redis` | string | Redis cache status (used for rate limiting and caching) |
| `services.nagl` | string | NAGL classification engine status |
| `timestamp` | string | ISO 8601 timestamp of the health check |
| `latency_ms` | number | Time taken to complete all health checks in milliseconds |

<Note>
  For real-time status monitoring, visit [status.trynawa.com](https://status.trynawa.com).
</Note>
