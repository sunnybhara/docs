Check the operational status of the NAWA API. This endpoint is **free** and does **not** require authentication.

<Note>
  The health endpoint returns a raw JSON response without the standard `{success, result, errors, request_id}` envelope.
</Note>

## Request

<CodeGroup>

```bash cURL
curl https://api.trynawa.com/v1/health
```

```python Python
import requests

response = requests.get("https://api.trynawa.com/v1/health")
data = response.json()
print(data["status"])  # 'healthy'
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/health')
const data = await response.json()
console.log(data.status)  // 'healthy'
```

</CodeGroup>

## Response

### Healthy response (200)

```json
{
  "status": "healthy",
  "version": "v1",
  "services": {
    "database": {
      "status": "healthy",
      "latency_ms": 12
    },
    "redis": {
      "status": "healthy",
      "latency_ms": 3
    },
    "nagl": {
      "status": "healthy",
      "latency_ms": 45
    }
  },
  "timestamp": "2025-01-15T12:00:00Z",
  "latency_ms": 52
}
```

### Degraded response (503)

```json
{
  "status": "degraded",
  "version": "v1",
  "services": {
    "database": {
      "status": "healthy",
      "latency_ms": 15
    },
    "redis": {
      "status": "down",
      "latency_ms": null
    },
    "nagl": {
      "status": "healthy",
      "latency_ms": 50
    }
  },
  "timestamp": "2025-01-15T12:00:00Z",
  "latency_ms": 65
}
```

### Response fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Overall status: `healthy` or `degraded` |
| `version` | string | API version |
| `services.database` | object | PostgreSQL status and latency |
| `services.redis` | object | Redis cache status and latency |
| `services.nagl` | object | NAGL classification engine status and latency |
| `timestamp` | string | ISO 8601 timestamp |
| `latency_ms` | integer | Total health check latency |

Each service object contains:
- `status`: `healthy`, `degraded`, or `down`
- `latency_ms`: Response time in milliseconds (null if down)

<Note>
  For real-time status monitoring, visit [status.trynawa.com](https://status.trynawa.com).
</Note>
