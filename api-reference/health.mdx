Check the operational status of the NAWA API and its dependencies. This endpoint is **free** and does **not** require authentication.

<Note>
  The health endpoint does not use the standard response envelope. It returns a flat JSON object.
</Note>

## Request

<CodeGroup>

```bash cURL
curl https://api.trynawa.com/v1/health
```

```python Python
import requests

response = requests.get("https://api.trynawa.com/v1/health")
result = response.json()
```

```typescript TypeScript
const response = await fetch("https://api.trynawa.com/v1/health");
const result = await response.json();
```

</CodeGroup>

## Response

### Healthy response (200)

```json
{
  "status": "healthy",
  "version": "v1",
  "services": {
    "database": {
      "status": "healthy",
      "latency_ms": 12
    },
    "redis": {
      "status": "healthy",
      "latency_ms": 3
    },
    "nagl": {
      "status": "healthy",
      "latency_ms": 1
    }
  },
  "timestamp": "2025-01-15T12:00:00Z",
  "latency_ms": 15
}
```

### Degraded response (503)

When one or more services are unhealthy, the endpoint returns HTTP 503:

```json
{
  "status": "degraded",
  "version": "v1",
  "services": {
    "database": {
      "status": "healthy",
      "latency_ms": 14
    },
    "redis": {
      "status": "degraded",
      "latency_ms": 250
    },
    "nagl": {
      "status": "healthy",
      "latency_ms": 2
    }
  },
  "timestamp": "2025-01-15T12:00:00Z",
  "latency_ms": 265
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `status` | string | Overall status: `healthy` or `degraded` |
| `version` | string | API version (always `v1`) |
| `services` | object | Individual service health checks |
| `services.[name].status` | string | `healthy`, `degraded`, or `down` |
| `services.[name].latency_ms` | integer | Service response time in milliseconds |
| `timestamp` | string | ISO 8601 timestamp of the health check |
| `latency_ms` | integer | Total health check duration in milliseconds |

<Note>
  For real-time status monitoring, visit [status.trynawa.com](https://status.trynawa.com).
</Note>
