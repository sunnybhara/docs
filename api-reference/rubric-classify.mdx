Classify a comment against a custom rubric. Define your own categories with descriptions for domain-specific classification.

<Note>
  Cost: **$0.003** per request (3 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

## Request

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify. |
| `rubric` | object | Yes | The rubric definition. |
| `rubric.categories` | array | Yes | Array of category objects. Max 20 categories. |
| `rubric.categories[].name` | string | Yes | Category name. |
| `rubric.categories[].description` | string | No | Description to guide the model on what this category means. |
| `rubric.multi_label` | boolean | No | If `true`, return all matching categories above the confidence threshold. Default: `false` (single best match). |
| `rubric.confidence_threshold` | number | No | Minimum confidence score (0-1) for a category match. Default: `0.5`. |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, `allam`. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/rubric/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "وين الترجمة العربية؟ مافهمت شي",
    "rubric": {
      "categories": [

        { "name": "off_topic", "description": "Not related to the video" }
      ],
      "multi_label": false,
      "confidence_threshold": 0.5
    }
  }'
```

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "id": "rcl_nw_abc123def456",
    "object": "rubric_classification",
    "categories": [
      { "name": "translation_request", "confidence": 0.94 }
    ],
    "multi_label": false,
    "language": "ar",
    "provider": "allam",
    "model": "sdaia/allam-1-13b-instruct",
    "fallback_used": false,
    "cost_usd": 0.003,
    "credits_used": 3
  },
  "errors": [],
  "request_id": "req_rub123abc456"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique classification ID (prefixed `rcl_`) |
| `object` | string | Always `"rubric_classification"` |
| `categories` | array | Array of matched categories with `name` and `confidence` (0-1) |
| `multi_label` | boolean | Whether multi-label mode was used |
| `language` | string | Detected language code (`ar`, `en`) |
| `provider` | string | AI provider used (`allam`, `claude`, `gemini`) |
| `model` | string | Specific model version used |
| `fallback_used` | boolean | Whether the fallback provider was used |
| `cost_usd` | number | Cost in USD |
| `credits_used` | number | Credits deducted |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or `rubric`, empty categories, exceeds 20 categories, category missing `name` |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 405 | `method_not_allowed` | Using a method other than POST |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
