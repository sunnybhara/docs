Classify text against a custom rubric. Define your own categories for domain-specific classification with configurable confidence thresholds and multi-label support.

<Note>
  Cost: **$0.003** per request (3 credits). **500 free requests/month** on this endpoint for all tiers.
</Note>

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Minimum 1 character. |
| `rubric` | object | Yes | The rubric definition. |
| `rubric.categories` | object[] | Yes | Array of category objects. Max 20 categories. Each must have a `name` field and optional `description`. |
| `rubric.multi_label` | boolean | No | Allow multiple categories to match. Default: `false`. |
| `rubric.confidence_threshold` | number | No | Minimum confidence to include a category (0.0-1.0). Default: `0.5`. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/rubric-classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "وين الترجمة العربية؟ مافهمت شي",
    "rubric": {
      "categories": [
        { "name": "translation_request", "description": "User is asking for subtitles or translation" },
        { "name": "technical_issue", "description": "Audio, video, or playback problems" },
        { "name": "content_feedback", "description": "Comments about the content quality" },
        { "name": "off_topic" }
      ],
      "multi_label": false,
      "confidence_threshold": 0.5
    }
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.rubric.classify({

  rubric: {

      { name: 'technical_issue', description: 'Audio, video, or playback problems' },
      { name: 'content_feedback', description: 'Comments about the content quality' },
      { name: 'off_topic' }
    ],
    multiLabel: false,
    confidenceThreshold: 0.5
  }
})
```

```python Python

            {"name": "off_topic"},

        "multi_label": False,

    },

```


### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`rcl_nw_` prefix) |
| `object` | string | Always `rubric_classification` |
| `categories` | object[] | Each category with `name` and `confidence` (0-1) |
| `multi_label` | boolean | Whether multi-label mode was used |
| `language` | string | Detected language: `ar`, `en`, or `mixed` |
| `provider` | string | AI provider used |
| `model` | string | Model version used |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits consumed |
