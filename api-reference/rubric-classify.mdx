Classify text against your own rubric. Define custom categories with optional descriptions, enable multi-label classification, and set confidence thresholds. Results are filtered by the threshold you specify.

<Note>
  Cost: **$0.003** per request (3 credits). No semantic caching on this endpoint.
</Note>

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Max 5,000 characters. |
| `rubric` | object | Yes | The rubric definition (see below). |
| `rubric.categories` | object[] | Yes | Array of category objects. Each must have a `name` (string). Optional `description` (string) to guide the model. 1-20 items. |
| `rubric.multi_label` | boolean | No | Allow multiple categories to match. Default: `false`. |
| `rubric.confidence_threshold` | number | No | Minimum confidence to include a category (0-1). Default: `0.5`. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/rubric-classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "وين الترجمة العربية؟ مافهمت شي",
    "rubric": {
      "categories": [
        {"name": "translation_request", "description": "User is asking for subtitles or translation"},
        {"name": "technical_issue", "description": "Audio, video, or playback problems"},
        {"name": "content_feedback", "description": "Comments about the content quality"},
        {"name": "off_topic", "description": "Not related to the video"}
      ],
      "multi_label": false,
      "confidence_threshold": 0.5
    }
  }'
```

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "id": "rcl_nw_a1b2c3d4e5f6",
    "object": "rubric_classification",
    "categories": [
      {"name": "translation_request", "confidence": 0.94},
      {"name": "technical_issue", "confidence": 0.03}
    ],
    "multi_label": false,
    "language": "ar",
    "provider": "allam",
    "model": "sdaia/allam-1-13b-instruct",
    "fallback_used": false,
    "cost_usd": 0.003,
    "credits_used": 3
  },
  "errors": [],
  "request_id": "req_nw_rcl1a2b3c4d5e"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID in `rcl_nw_xxx` format |
| `object` | string | Always `"rubric_classification"` |
| `categories` | object[] | Array of `{name, confidence}` objects, sorted by confidence descending. Filtered by `confidence_threshold`. |
| `multi_label` | boolean | Whether multi-label mode was used |
| `language` | string | `ar`, `en`, or `mixed` |
| `provider` | string | AI provider used |
| `model` | string | Model version |
| `fallback_used` | boolean | Whether the fallback provider was used |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits consumed |
