Classify text against a custom rubric you define. Supports multi-label classification and configurable confidence thresholds for domain-specific use cases.

<Note>
  Cost: **3 credits ($0.003)** per request. Free tier keys (sandbox) are not charged credits.
</Note>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, or `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Must be non-empty. |
| `rubric` | object | Yes | The rubric definition. |
| `rubric.categories` | array | Yes | Array of category objects. Each must have a `name` (string). Optional `description` (string) per category. Max 20 categories. |
| `rubric.multi_label` | boolean | No | Allow multiple matching categories. Default: `false`. |
| `rubric.confidence_threshold` | number | No | Minimum confidence score (0 to 1) for a category to be included. Default: `0.5`. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/rubric-classify \
  -H "Authorization: Bearer nawa_live_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "وين الترجمة العربية؟ مافهمت شي",
    "rubric": {
      "categories": [
        {"name": "translation_request", "description": "User is asking for subtitles or translation"},
        {"name": "technical_issue", "description": "Audio, video, or playback problems"},
        {"name": "content_feedback", "description": "Comments about the content quality"},
        {"name": "off_topic", "description": "Not related to the video"}
      ],
      "multi_label": false,
      "confidence_threshold": 0.5
    }
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const result = await nawa.rubric.classify({
  text: 'وين الترجمة العربية؟ مافهمت شي',
  rubric: {
    categories: [
      { name: 'translation_request', description: 'User is asking for subtitles or translation' },
      { name: 'technical_issue', description: 'Audio, video, or playback problems' },
      { name: 'content_feedback', description: 'Comments about the content quality' },
      { name: 'off_topic', description: 'Not related to the video' }
    ],
    multiLabel: false,
    confidenceThreshold: 0.5
  }
})

console.log(result.categories) // [{ name: "translation_request", confidence: 0.94 }]
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="nawa_live_sk_xxx")

result = nawa.rubric.classify(
    text="وين الترجمة العربية؟ مافهمت شي",
    rubric={
        "categories": [
            {"name": "translation_request", "description": "User is asking for subtitles or translation"},
            {"name": "technical_issue", "description": "Audio, video, or playback problems"},
            {"name": "content_feedback", "description": "Comments about the content quality"},
            {"name": "off_topic", "description": "Not related to the video"},
        ],
        "multi_label": False,
        "confidence_threshold": 0.5,
    },
)

print(result.categories)  # [{"name": "translation_request", "confidence": 0.94}]
```

## Response

### Success response (200)

```json
{
  "result": {
    "id": "rcl_nw_a1b2c3d4e5f6",
    "object": "rubric_classification",
    "categories": [
      { "name": "translation_request", "confidence": 0.94 }
    ],
    "multi_label": false,
    "language": "ar",
    "provider": "allam",
    "model": "sdaia/allam-1-13b-instruct",
    "fallback_used": false,
    "cost_usd": 0.003,
    "credits_used": 3
  },
  "success": true,
  "errors": [],
  "request_id": "req_nw_a1b2c3d4e5f67890abcdef12"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Rubric classification ID (`rcl_nw_xxx` format) |
| `object` | string | Always `"rubric_classification"` |
| `categories` | array | Matched categories with `name` and `confidence` (0 to 1, rounded to 2 decimal places). Only categories above the `confidence_threshold` are returned. |
| `multi_label` | boolean | Whether multi-label mode was used |
| `language` | string | Detected language code |
| `provider` | string | AI provider used |
| `model` | string | Specific model version |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits deducted (0 for sandbox keys) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or `rubric`, invalid categories, exceeds 20 categories |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `permission_error` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
