Classify text against a user-defined rubric. Define your own categories with optional descriptions, enable multi-label classification, and set confidence thresholds. Available on the free tier.

<Note>
  Cost: **$0.003** per request (3 credits). This endpoint is available on the free tier.
</Note>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, `allam` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify (min 1 character). |
| `rubric` | object | Yes | The rubric definition (see below). |
| `rubric.categories` | array | Yes | Array of category objects. Each has a required `name` (string) and optional `description` (string). Max 20 categories. |
| `rubric.multi_label` | boolean | No | Allow multiple category matches. Default: `false`. |
| `rubric.confidence_threshold` | number | No | Minimum confidence to include a category (0-1). Default: `0.5`. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST "https://api.trynawa.com/v1/rubric-classify" \
  -H "Authorization: Bearer nawa_live_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "أحتاج مساعدة في طلبي رقم ١٢٣٤",
    "rubric": {
      "categories": [
        { "name": "order_support", "description": "Questions about orders" },
        { "name": "technical_issue", "description": "Technical problems" },
        { "name": "billing", "description": "Payment and billing inquiries" },
        { "name": "general_feedback", "description": "General comments" }
      ],
      "multi_label": false,
      "confidence_threshold": 0.5
    }
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.rubric.classify({
  text: 'أحتاج مساعدة في طلبي رقم ١٢٣٤',
  rubric: {
    categories: [
      { name: 'order_support', description: 'Questions about orders' },
      { name: 'technical_issue', description: 'Technical problems' },
      { name: 'billing', description: 'Payment and billing inquiries' },
      { name: 'general_feedback', description: 'General comments' }
    ],
    multiLabel: false,
    confidenceThreshold: 0.5
  }
})

if (data) {
  console.log(data.categories) // [{ name: "order_support", confidence: 0.94 }]
}
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.rubric.classify(
    text="أحتاج مساعدة في طلبي رقم ١٢٣٤",
    rubric={
        "categories": [
            {"name": "order_support", "description": "Questions about orders"},
            {"name": "technical_issue", "description": "Technical problems"},
            {"name": "billing", "description": "Payment and billing inquiries"},
            {"name": "general_feedback", "description": "General comments"},
        ],
        "multi_label": False,
        "confidence_threshold": 0.5,
    },
)

if result.data:
    print(result.data.categories)  # [{"name": "order_support", "confidence": 0.94}]
```

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "id": "rcl_nw_a1b2c3d4e5f6",
    "object": "rubric_classification",
    "categories": [
      { "name": "order_support", "confidence": 0.94 },
      { "name": "billing", "confidence": 0.12 },
      { "name": "technical_issue", "confidence": 0.05 },
      { "name": "general_feedback", "confidence": 0.02 }
    ],
    "multi_label": false,
    "language": "ar",
    "provider": "allam",
    "model": "sdaia/allam-1-13b-instruct",
    "fallback_used": false,
    "cost": 0.003
  },
  "errors": [],
  "request_id": "req_nw_a1b2c3d4e5f67890abcdef12"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`rcl_nw_xxx` format) |
| `object` | string | Always `rubric_classification` |
| `categories` | array | Array of `{ name, confidence }` objects, sorted by confidence descending |
| `multi_label` | boolean | Whether multi-label mode was used |
| `language` | string | Detected language code |
| `provider` | string | AI provider used |
| `model` | string | Specific model version |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `cost` | number | Cost in USD |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or `rubric`, invalid categories |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `permission_error` | Insufficient credits |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
