Generate a structured audience intelligence report from comment data. The report will analyze audience clusters, dominant narratives, sentiment patterns, spam/bot detection, and produce data-backed strategic recommendations.

## Planned pricing

| Tier | Cost | Sections |
|------|------|----------|
| Basic | $0.50 (500 credits) | Executive summary, audience clusters, narrative map, sentiment breakdown, top commenters, spam detection, recommendations, channel health |
| Pro | $1.50 (1500 credits) | Everything in Basic, plus repeat commenter analysis and content brief |

## Planned request format

```json
{
  "comments": [
    {"text": "This is the best explanation of AI I have seen", "author": "TechFan42", "likes": 156},
    {"text": "Can you do a follow up on the regulation angle?", "author": "PolicyNerd", "likes": 45}
  ],
  "tier": "pro",
  "title": "AI Is Coming For Your Job",
  "channel_context": "Tech commentary channel, 500K subscribers"
}
```

### Planned parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `comments` | array | Yes | Array of comment objects. Min 10, max 500. Each must have a `text` field. |
| `comments[].text` | string | Yes | The comment text. |
| `comments[].author` | string | No | Commenter handle or name. |
| `comments[].likes` | integer | No | Number of likes on the comment. |
| `comments[].created_at` | string | No | ISO 8601 timestamp. |
| `tier` | string | No | `basic` or `pro`. Default: `basic`. |
| `title` | string | No | Video or content title for context. |
| `channel_context` | string | No | Brief channel description. |
