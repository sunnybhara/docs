Retrieve aggregated analytics for your API key including request counts, costs, latency, cache performance, and error rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group results by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const { data, error } = await nawa.analytics.get({
  from: '2025-01-01T00:00:00Z',
  to: '2025-01-31T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
result = nawa.analytics.get(
    from_date="2025-01-01T00:00:00Z",
    to_date="2025-01-31T23:59:59Z",
    group_by="endpoint"
)
```

</CodeGroup>

<Note>
  The breakdown key name changes based on `group_by`: `by_endpoint`, `by_day`, or `by_provider`. Each item in the array uses a matching key (`endpoint`, `day`, or `provider`) for its label.
</Note>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of requests served from cache |
| `analytics.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `analytics.avg_latency_ms` | integer | Average response latency in milliseconds |
| `analytics.success_rate` | number | Success rate (0-1) |
| `analytics.error_count` | integer | Total errors in the period |
| `analytics.endpoint_distribution` | object | Request counts per endpoint |
| `analytics.provider_distribution` | object | Request counts per AI provider |
| `analytics.period` | object | The queried time range |
| `by_endpoint` | array | Breakdown with `requests`, `cost`, `avg_latency_ms` per group |
