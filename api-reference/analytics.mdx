Retrieve aggregated analytics across your API usage. Includes request counts, costs, cache hit rates, latency, and breakdowns by endpoint, day, or provider. This endpoint is **free**.

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). |
| `to` | string | No | End date (ISO 8601). |
| `group_by` | string | No | Group results by: `endpoint`, `day`, or `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const params = new URLSearchParams({
  from: '2025-01-01T00:00:00Z',
  group_by: 'endpoint'
})

const response = await fetch(`https://api.trynawa.com/v1/analytics?${params}`, {
  headers: {
    'Authorization': 'Bearer nawa_test_sk_xxx'
  }
})

const data = await response.json()
console.log(data.result.analytics.total_requests)
console.log(data.result.analytics.cache_hit_rate)
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/analytics",
    headers={"Authorization": "Bearer nawa_test_sk_xxx"},
    params={
        "from": "2025-01-01T00:00:00Z",
        "group_by": "endpoint"
    }
)

data = response.json()
print(data["result"]["analytics"]["total_requests"])
print(data["result"]["analytics"]["cache_hit_rate"])
```

</CodeGroup>

<Note>
  The breakdown key changes based on `group_by`: `by_endpoint`, `by_day`, or `by_provider`. When grouped by day, each item includes a `date` field (`YYYY-MM-DD`) instead of `endpoint`.
</Note>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of semantic cache hits |
| `analytics.cache_hit_rate` | number | Cache hit percentage (0-1) |
| `analytics.avg_latency_ms` | integer | Average response latency in milliseconds |
| `analytics.success_rate` | number | Percentage of successful requests (0-1) |
| `analytics.error_count` | integer | Number of failed requests |
| `analytics.endpoint_distribution` | object | Request counts by endpoint |
| `analytics.provider_distribution` | object | Request counts by AI provider |
| `analytics.period` | object | The time range queried (`from` and `to`) |
| `by_endpoint` | object[] | Per-endpoint breakdown with `requests`, `cost`, `avg_latency_ms` |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Invalid `group_by` value |
| 401 | `authentication_error` | Invalid or missing API key |
| 405 | `invalid_request_error` | Non-GET request method |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
