Retrieve aggregated analytics across your API usage. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601): `2025-01-01T00:00:00Z` |
| `to` | string | No | End date (ISO 8601): `2025-01-31T23:59:59Z` |
| `group_by` | string | No | Group results by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

// Analytics are available through the REST API
const response = await fetch(
  'https://api.trynawa.com/v1/analytics?group_by=endpoint',
  { headers: { Authorization: `Bearer ${process.env.NAWA_API_KEY}` } }
)
const data = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/analytics",
    headers={"Authorization": "Bearer nawa_test_sk_xxx"},
    params={"group_by": "endpoint"}
)
data = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of semantic cache hits |
| `analytics.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `analytics.avg_latency_ms` | integer | Average server-side latency |
| `analytics.success_rate` | number | Success ratio (0-1) |
| `analytics.error_count` | integer | Total error responses |
| `analytics.endpoint_distribution` | object | Request count per endpoint |
| `analytics.provider_distribution` | object | Request count per AI provider |
| `analytics.period` | object | Requested time period |
| `by_endpoint` / `by_day` / `by_provider` | array | Breakdown grouped by the `group_by` parameter |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Invalid `group_by` value |
| 401 | `authentication_error` | Invalid or missing API key |
| 405 | `method_not_allowed` | Using a method other than GET |
| 500 | `api_error` | Internal error |
