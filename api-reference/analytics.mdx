Retrieve aggregated API usage analytics across your account. Includes request counts, costs, cache hit rates, latency, and breakdowns by endpoint or provider. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group results by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?group_by=endpoint" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="nawa_live_sk_xxx")

result = nawa.analytics.get(group_by="endpoint")

print(result.analytics.total_requests)  # 12450
print(result.analytics.cache_hit_rate)  # 0.23
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.analytics.get({
  groupBy: 'endpoint'
})

console.log(data.analytics.totalRequests)  // 12450
console.log(data.analytics.cacheHitRate)   // 0.23
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Total semantic cache hits |
| `analytics.cache_hit_rate` | number | Cache hit rate (0--1) |
| `analytics.avg_latency_ms` | integer | Average response latency in milliseconds |
| `analytics.success_rate` | number | Success rate (0--1) |
| `analytics.error_count` | integer | Total error responses |
| `by_endpoint[].endpoint` | string | API endpoint path |
| `by_endpoint[].requests` | integer | Request count for this endpoint |
| `by_endpoint[].cost` | number | Cost in USD for this endpoint |
| `by_endpoint[].cache_hits` | integer | Cache hits for this endpoint |
