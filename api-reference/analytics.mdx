Retrieve aggregated analytics across your API requests including cost breakdowns, cache hit rates, and latency metrics. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group results by: `endpoint`, `day`, or `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.analytics({
  from: '2025-01-01T00:00:00Z',
  to: '2025-01-31T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.analytics(
    from_date="2025-01-01T00:00:00Z",
    to_date="2025-01-31T23:59:59Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total credit cost in USD |
| `analytics.cache_hits` | integer | Number of requests served from cache |
| `analytics.cache_hit_rate` | number | Ratio of cache hits to total requests (0-1) |
| `analytics.avg_latency_ms` | number | Average response time in milliseconds |
| `analytics.success_rate` | number | Ratio of successful requests (0-1) |
| `analytics.error_count` | integer | Number of failed requests |
| `analytics.endpoint_distribution` | object | Request count per endpoint |
| `analytics.provider_distribution` | object | Request count per AI provider |
| `analytics.period` | object | The queried time range |
| `by_endpoint` / `by_day` / `by_provider` | array | Breakdown grouped by the `group_by` parameter |
