Retrieve aggregated analytics across your API usage, including request counts, costs, latency, cache performance, and provider distribution. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group results by: `endpoint`, `day`, or `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-03-01T00:00:00Z&to=2026-03-17T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.analytics.get({
  from: '2026-03-01T00:00:00Z',
  to: '2026-03-17T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.analytics.get(
    from_date="2026-03-01T00:00:00Z",
    to_date="2026-03-17T23:59:59Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total credit cost in USD |
| `analytics.cache_hits` | integer | Number of cache hits |
| `analytics.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `analytics.avg_latency_ms` | integer | Average response latency in milliseconds |
| `analytics.success_rate` | number | Success ratio (0-1) |
| `analytics.error_count` | integer | Number of error responses |
| `analytics.endpoint_distribution` | object | Request counts by endpoint |
| `analytics.provider_distribution` | object | Request counts by AI provider |
| `analytics.period` | object | The `from` and `to` date range queried |
| `by_endpoint` | array | Breakdown grouped by the `group_by` parameter |
