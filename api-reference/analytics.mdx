Retrieve aggregated API analytics including request counts, costs, cache efficiency, latency, and success rates. Results are scoped to your API key. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `from` | string | No | -- | Start date (ISO 8601) |
| `to` | string | No | -- | End date (ISO 8601) |
| `group_by` | string | No | `endpoint` | Group results by: `endpoint`, `day`, `provider` |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const result = await nawa.analytics.get({
  from: '2025-01-01T00:00:00Z',
  to: '2025-01-31T23:59:59Z',
  groupBy: 'endpoint'
})

console.log(result.data.analytics.total_requests)
console.log(result.data.analytics.cache_hit_rate)
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.analytics.get(
    from_date="2025-01-01T00:00:00Z",
    to_date="2025-01-31T23:59:59Z",
    group_by="endpoint"
)

print(result.data.analytics.total_requests)
print(result.data.analytics.cache_hit_rate)
```

</CodeGroup>

</Note>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of cache hits |
| `analytics.cache_hit_rate` | number | Cache hit rate 0-1 |
| `analytics.avg_latency_ms` | number | Average response latency in milliseconds |
| `analytics.success_rate` | number | Request success rate 0-1 |
| `analytics.error_count` | integer | Total error count |
| `analytics.endpoint_distribution` | object | Request counts by endpoint |
| `analytics.provider_distribution` | object | Request counts by provider |
| `analytics.period` | object | The time period with `from` and `to` |
| `by_endpoint` | array | Breakdown by endpoint (when `group_by=endpoint`) |
| `by_day` | array | Breakdown by day (when `group_by=day`) |
| `by_provider` | array | Breakdown by provider (when `group_by=provider`) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
