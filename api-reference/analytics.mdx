Retrieve aggregated analytics for your API usage including request counts, costs, cache hit rates, average latency, success rates, and provider distribution. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: all time. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group results by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-01-01T00:00:00Z&to=2026-03-17T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

// Use a raw fetch since the SDK may not wrap this endpoint
const response = await fetch('https://api.trynawa.com/v1/analytics?group_by=endpoint', {
  headers: { Authorization: `Bearer ${process.env.NAWA_API_KEY}` }
})
const data = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/analytics",
    params={"group_by": "endpoint"},
    headers={"Authorization": "Bearer your_api_key"}
)
data = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total credit cost in USD |
| `analytics.cache_hits` | integer | Number of cache hit requests |
| `analytics.cache_hit_rate` | number | Ratio of cache hits to total requests (0--1) |
| `analytics.avg_latency_ms` | integer | Average response latency in milliseconds |
| `analytics.success_rate` | number | Ratio of 2xx responses to total (0--1) |
| `analytics.error_count` | integer | Number of 4xx/5xx responses |
| `analytics.endpoint_distribution` | object | Request count per endpoint |
| `analytics.provider_distribution` | object | Request count per AI provider |
| `analytics.period` | object | The time period queried |
| `by_endpoint` / `by_day` / `by_provider` | array | Grouped breakdown (depends on `group_by`) |
