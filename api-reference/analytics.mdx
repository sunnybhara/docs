Retrieve aggregated analytics across your API usage. This endpoint is **free**.

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: 30 days ago. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group results by: `endpoint`, `day`, `provider`. Default: none (aggregate only). |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-01-01T00:00:00Z&to=2026-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer $NAWA_API_KEY"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.analytics({
  from: '2026-01-01T00:00:00Z',
  to: '2026-01-31T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.analytics(
    from_date="2026-01-01T00:00:00Z",
    to_date="2026-01-31T23:59:59Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total credit cost in USD |
| `analytics.cache_hits` | integer | Number of semantic cache hits |
| `analytics.cache_hit_rate` | number | Ratio of cache hits to total requests (0-1) |
| `analytics.avg_latency_ms` | number | Average response latency in milliseconds |
| `analytics.success_rate` | number | Ratio of successful requests (0-1) |
| `analytics.error_count` | integer | Total failed requests |
| `analytics.endpoint_distribution` | object | Request count by endpoint |
| `analytics.provider_distribution` | object | Request count by AI provider |
| `analytics.period` | object | Start and end dates for the query |
| `by_endpoint` | array | Breakdown by endpoint (when `group_by=endpoint`) |
| `by_day` | array | Breakdown by day (when `group_by=day`) |
| `by_provider` | array | Breakdown by provider (when `group_by=provider`) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
