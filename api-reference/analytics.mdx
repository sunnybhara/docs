Retrieve aggregated analytics across your classified content. Group by day, endpoint, or provider. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601), e.g. `2026-01-01T00:00:00Z`. |
| `to` | string | No | End date (ISO 8601), e.g. `2026-01-31T23:59:59Z`. |
| `group_by` | string | No | Group results by: `endpoint`, `day`, or `provider`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-01-01T00:00:00Z&to=2026-01-31T23:59:59Z&group_by=day" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.analytics({
  from: '2026-01-01T00:00:00Z',
  to: '2026-01-31T23:59:59Z',
  groupBy: 'day'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.analytics(
    from_date="2026-01-01T00:00:00Z",
    to_date="2026-01-31T23:59:59Z",
    group_by="day",
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of cache hits |
| `analytics.cache_hit_rate` | number | Ratio of cache hits to total requests (0-1) |
| `analytics.avg_latency_ms` | number | Average response latency in milliseconds |
| `analytics.success_rate` | number | Ratio of successful requests (0-1) |
| `analytics.error_count` | integer | Total error count |
| `analytics.endpoint_distribution` | object | Request counts grouped by endpoint |
| `analytics.provider_distribution` | object | Request counts grouped by AI provider |
| `analytics.period` | object | The `from` and `to` timestamps for the query |
| `by_endpoint` / `by_day` / `by_provider` | array | Breakdown array (varies based on `group_by`) |
