Retrieve aggregated analytics on your API usage including request counts, costs, cache hit rates, latency, and success rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group results by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/analytics",
    headers={"Authorization": "Bearer nawa_live_sk_xxx"},
    params={
        "from": "2025-01-01T00:00:00Z",
        "to": "2025-01-31T23:59:59Z",
        "group_by": "endpoint",
    },
)
result = response.json()
```

```typescript TypeScript
const params = new URLSearchParams({
  from: "2025-01-01T00:00:00Z",
  to: "2025-01-31T23:59:59Z",
  group_by: "endpoint",
});

const response = await fetch(
  `https://api.trynawa.com/v1/analytics?${params}`,
  {
    headers: { Authorization: "Bearer nawa_live_sk_xxx" },
  }
);
const result = await response.json();
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of cache hits |
| `analytics.cache_hit_rate` | number | Cache hit rate (0--1) |
| `analytics.avg_latency_ms` | integer | Average latency in milliseconds |
| `analytics.success_rate` | number | Success rate (0--1) |
| `analytics.error_count` | integer | Total error count |
| `analytics.endpoint_distribution` | object | Request counts per endpoint |
| `analytics.provider_distribution` | object | Request counts per AI provider |
| `analytics.period` | object | The `from` and `to` timestamps for the query |
| `by_[group_by]` | array | Breakdown grouped by your `group_by` parameter |

<Note>
  When you use `group_by=provider`, the response includes a `by_provider` array instead of `by_endpoint`.
</Note>

### Error responses

| Status | Type | When |
|--------|------|------|
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
