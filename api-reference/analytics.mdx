Retrieve aggregated analytics on your API usage including request counts, costs, cache performance, and provider distribution. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group breakdown by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.analytics({
  from: '2025-01-01T00:00:00Z',
  to: '2025-01-31T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.analytics(
    from_date="2025-01-01T00:00:00Z",
    to_date="2025-01-31T23:59:59Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Analytics fields

| Field | Type | Description |
|-------|------|-------------|
| `total_requests` | integer | Total API requests in the period |
| `total_cost` | number | Total cost in USD |
| `cache_hits` | integer | Number of cache hits |
| `cache_hit_rate` | number | Cache hit ratio (0-1) |
| `avg_latency_ms` | integer | Average response time in milliseconds |
| `success_rate` | number | Ratio of successful requests (0-1) |
| `error_count` | integer | Number of failed requests |
| `endpoint_distribution` | object | Request counts per endpoint |
| `provider_distribution` | object | Request counts per AI provider |
| `period` | object | Contains `from` and `to` dates |

### Breakdown fields

The `by_{group_by}` array contains one entry per group:

| Field | Type | Description |
|-------|------|-------------|
| `endpoint` / `day` / `provider` | string | The group key (varies by `group_by` value) |
| `requests` | integer | Request count |
| `cost` | number | Total cost in USD |
| `avg_latency_ms` | integer | Average latency in ms |

### Grouping examples

<AccordionGroup>
  <Accordion title="Group by day">
```bash
curl "https://api.trynawa.com/v1/analytics?group_by=day&from=2025-01-01T00:00:00Z" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```
Returns `by_day` array with entries like `{"day": "2025-01-15", "requests": 450, "cost": 2.70, "avg_latency_ms": 800}`.
  </Accordion>

  <Accordion title="Group by provider">
```bash
curl "https://api.trynawa.com/v1/analytics?group_by=provider" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```
Returns `by_provider` array with entries like `{"provider": "allam", "requests": 7200, "cost": 43.20, "avg_latency_ms": 750}`.
  </Accordion>
</AccordionGroup>

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Invalid `group_by` value |
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
