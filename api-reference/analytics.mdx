Retrieve aggregated analytics for your API usage. Returns request counts, costs, cache hit rates, and breakdowns by endpoint, day, or provider. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). |
| `to` | string | No | End date (ISO 8601). |
| `group_by` | string | No | Group results by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-01-01T00:00:00Z&to=2026-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const params = new URLSearchParams({
  from: "2026-01-01T00:00:00Z",
  to: "2026-01-31T23:59:59Z",
  group_by: "endpoint",
});

const response = await fetch(
  `https://api.trynawa.com/v1/analytics?${params}`,
  {
    headers: {
      "Authorization": "Bearer nawa_test_sk_xxx",
    },
  },
);

const data = await response.json();
console.log(data.result.analytics);
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/analytics",
    headers={"Authorization": "Bearer nawa_test_sk_xxx"},
    params={
        "from": "2026-01-01T00:00:00Z",
        "to": "2026-01-31T23:59:59Z",
        "group_by": "endpoint",
    },
)

data = response.json()
print(data["result"]["analytics"])
```

</CodeGroup>

### Analytics fields

| Field | Type | Description |
|-------|------|-------------|
| `total_requests` | integer | Total API requests in the period |
| `total_cost` | number | Total cost in USD |
| `cache_hits` | integer | Number of requests served from cache |
| `cache_hit_rate` | number | Percentage of cache hits (0-1) |
| `avg_latency_ms` | integer | Average response latency in milliseconds |
| `success_rate` | number | Percentage of successful requests (0-1) |
| `error_count` | integer | Number of requests that returned errors |
| `endpoint_distribution` | object | Request count per endpoint |
| `provider_distribution` | object | Request count per AI provider |
| `period` | object | The `from` and `to` timestamps for the query |

### Breakdown fields

The `by_{group_by}` array contains per-group breakdowns sorted by request count (descending):

| Field | Type | Description |
|-------|------|-------------|
| `endpoint` / `day` / `provider` | string | The group key (varies by `group_by` parameter) |
| `requests` | integer | Total requests in this group |
| `cost` | number | Total cost in USD |
| `avg_latency_ms` | integer | Average latency for this group |
