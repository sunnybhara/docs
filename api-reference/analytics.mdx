Retrieve aggregated analytics for your API usage. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group results by: `endpoint`, `day`, `provider`. Default: none (totals only). |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-03-01T00:00:00Z&to=2026-03-18T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.analytics({
  from: '2026-03-01T00:00:00Z',
  to: '2026-03-18T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.analytics(
    from_date="2026-03-01T00:00:00Z",
    to_date="2026-03-18T23:59:59Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of semantic cache hits |
| `analytics.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `analytics.avg_latency_ms` | integer | Average response time in milliseconds |
| `analytics.success_rate` | number | Success ratio (0-1) |
| `analytics.error_count` | integer | Total errors in the period |
| `analytics.endpoint_distribution` | object | Request counts by endpoint |
| `analytics.provider_distribution` | object | Request counts by AI provider |
| `analytics.period` | object | The queried time range |
| `by_[group_by]` | array | Breakdown by the requested grouping (when `group_by` is specified) |
