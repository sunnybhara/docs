Retrieve aggregated analytics across your API usage. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group breakdown by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const response = await fetch('https://api.trynawa.com/v1/analytics?group_by=endpoint', {
  headers: { 'Authorization': `Bearer ${process.env.NAWA_API_KEY}` }
})
const data = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/analytics",
    params={"group_by": "endpoint"},
    headers={"Authorization": "Bearer your_api_key"}
)
data = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of cache hits |
| `analytics.cache_hit_rate` | number | Cache hit rate (0-1) |
| `analytics.avg_latency_ms` | number | Average request latency in ms |
| `analytics.success_rate` | number | Rate of 2xx responses (0-1) |
| `analytics.error_count` | integer | Number of 4xx/5xx responses |
| `analytics.endpoint_distribution` | object | Request count by endpoint |
| `analytics.provider_distribution` | object | Request count by AI provider |
| `analytics.period` | object | Date range of the query |
| `by_<group_by>` | array | Breakdown grouped by the selected dimension |
