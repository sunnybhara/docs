Retrieve aggregated analytics for your API usage. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group results by: `endpoint`, `day`, or `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.analytics({
  from: '2025-01-01T00:00:00Z',
  to: '2025-01-31T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.analytics(
    from_date="2025-01-01T00:00:00Z",
    to_date="2025-01-31T23:59:59Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Analytics fields

| Field | Type | Description |
|-------|------|-------------|
| `total_requests` | integer | Total API requests in the period |
| `total_cost` | number | Total cost in USD |
| `cache_hits` | integer | Number of semantic cache hits |
| `cache_hit_rate` | number | Ratio of cache hits to total requests (0-1) |
| `avg_latency_ms` | number | Average response latency in milliseconds |
| `success_rate` | number | Ratio of successful requests (0-1) |
| `error_count` | integer | Total failed requests |
| `endpoint_distribution` | object | Request counts by endpoint |
| `provider_distribution` | object | Request counts by AI provider |
| `period` | object | The `from` and `to` dates of the query |

### Grouping options

When you set `group_by`, the response includes an additional array keyed by the group:

| `group_by` | Extra field | Description |
|------------|-------------|-------------|
| `endpoint` | `by_endpoint` | Breakdown by endpoint with `requests`, `cost`, `avg_latency_ms` |
| `day` | `by_day` | Breakdown by date |
| `provider` | `by_provider` | Breakdown by AI provider |
