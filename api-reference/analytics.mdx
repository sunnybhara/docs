Retrieve aggregated analytics across your API usage. Includes totals, cache hit rates, success rates, and breakdowns by endpoint, day, or provider. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group breakdown by: `endpoint`, `day`, `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-03-01T00:00:00Z&to=2026-03-24T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```typescript TypeScript
const response = await fetch(
  "https://api.trynawa.com/v1/analytics?from=2026-03-01T00:00:00Z&group_by=endpoint",
  {
    headers: { Authorization: "Bearer nawa_live_sk_xxx" },
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/analytics",
    headers={"Authorization": "Bearer nawa_live_sk_xxx"},
    params={
        "from": "2026-03-01T00:00:00Z",
        "group_by": "endpoint",
    },
)

data = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Total cache hits |
| `analytics.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `analytics.avg_latency_ms` | number | Average response time in milliseconds |
| `analytics.success_rate` | number | Success rate (0-1) |
| `analytics.error_count` | integer | Total error responses |
| `analytics.endpoint_distribution` | object | Request counts per endpoint |
| `analytics.provider_distribution` | object | Request counts per AI provider |
| `analytics.period` | object | The queried time period |
| `by_<group_by>` | array | Breakdown by the chosen dimension, sorted by request count descending |
