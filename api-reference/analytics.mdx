Retrieve aggregated analytics for your API usage, including request counts, costs, cache hit rates, and provider distribution. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601), e.g. `2026-01-01T00:00:00Z` |
| `to` | string | No | End date (ISO 8601), e.g. `2026-01-31T23:59:59Z` |
| `group_by` | string | No | Group results by: `endpoint`, `day`, or `provider`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-01-01T00:00:00Z&to=2026-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const { data, error } = await nawa.analytics({
  from: '2026-01-01T00:00:00Z',
  to: '2026-01-31T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
result = nawa.analytics(
    from_date="2026-01-01T00:00:00Z",
    to="2026-01-31T23:59:59Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `analytics.total_requests` | integer | Total API requests in the period |
| `analytics.total_cost` | number | Total cost in USD |
| `analytics.cache_hits` | integer | Number of semantic cache hits |
| `analytics.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `analytics.avg_latency_ms` | integer | Average response time in milliseconds |
| `analytics.success_rate` | number | Success ratio (0-1) |
| `analytics.error_count` | integer | Number of failed requests |
| `analytics.endpoint_distribution` | object | Request count per endpoint |
| `analytics.provider_distribution` | object | Request count per AI provider |
| `analytics.period` | object | Date range with `from` and `to` |
| `by_{group_by}` | array | Breakdown array grouped by your chosen dimension |
