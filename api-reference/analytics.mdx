Retrieve aggregated analytics across your classified requests. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601) |
| `to` | string | No | End date (ISO 8601) |
| `group_by` | string | No | Group results by: `day`, `endpoint`, or `provider`. Default: `day`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/analytics?from=2026-03-01T00:00:00Z&to=2026-03-17T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const response = await fetch(
  'https://api.trynawa.com/v1/analytics?group_by=endpoint',
  { headers: { Authorization: `Bearer ${process.env.NAWA_API_KEY}` } }
)
const analytics = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/analytics",
    params={"group_by": "endpoint"},
    headers={"Authorization": "Bearer your_api_key"}
)
analytics = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `total_requests` | integer | Total API requests in the period |
| `total_cost` | number | Total credit cost in USD |
| `cache_hit_rate` | number | Proportion of requests served from cache (0-1) |
| `avg_latency_ms` | integer | Average response latency in milliseconds |
| `success_rate` | number | Proportion of successful requests (0-1) |
| `error_count` | integer | Total error responses in the period |
| `by_endpoint` | object | Request counts grouped by endpoint |
| `by_provider` | object | Request counts grouped by AI provider |
