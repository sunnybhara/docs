Submit feedback on a classification result to improve NAWA's accuracy through reinforcement learning from human feedback (RLHF). Uses quorum-based pattern confirmation.

<Note>
  This endpoint is **free** for all tiers -- 0 credits, no cost.
</Note>

<Tip>
  Every feedback submission helps improve accuracy for all NAWA users. We especially value feedback on dialect detection edge cases.
</Tip>

## Request

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `classification_id` | string | Yes | The `cls_nw_xxx` ID from the original classify response. |
| `rating` | string | Yes | Your rating: `correct`, `incorrect`, or `partial`. |
| `corrected_intent` | string[] | No | The correct intent(s) the model should have returned. |
| `corrected_sentiment` | string | No | The correct sentiment: `positive`, `negative`, `neutral`, or `mixed`. |
| `comment` | string | No | Free-text explanation of why the correction is needed. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST "https://api.trynawa.com/v1/feedback" \
  -H "Authorization: Bearer nawa_live_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "corrected_intent": ["question"],
    "corrected_sentiment": "neutral",
    "comment": "This is a question about delivery, not a complaint"
  }'
```

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "id": "fb_nw_a1b2c3d4e5f6",
    "object": "feedback",
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "acknowledged": true
  },
  "errors": [],
  "request_id": "req_nw_a1b2c3d4e5f67890abcdef12"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Feedback ID in `fb_nw_xxx` format |
| `object` | string | Always `feedback` |
| `classification_id` | string | The classification this feedback refers to |
| `rating` | string | The submitted rating |
| `acknowledged` | boolean | Always `true` on success |
