Submit reinforcement learning from human feedback (RLHF) to improve NAWA classification accuracy. This endpoint is **free** for all tiers.

<Tip>
  Every feedback submission helps improve accuracy for all NAWA users. We especially value feedback on dialect detection edge cases.
</Tip>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `classification_id` | string | Yes | The `cls_nw_xxx` ID from the original classify response. |
| `rating` | string | Yes | Your rating: `correct`, `incorrect`, or `partial`. |
| `corrected_intent` | string[] | No | The correct intent(s) the model should have returned. |
| `corrected_sentiment` | string | No | The correct sentiment: `positive`, `negative`, `neutral`, or `mixed`. |
| `comment` | string | No | Free-text explanation of why this correction is needed. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/feedback \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "corrected_intent": ["question"],
    "corrected_sentiment": "neutral",
    "comment": "This is Lebanese Arabic, not Gulf"
  }'
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/feedback', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer nawa_test_sk_xxx',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    classification_id: 'cls_nw_a1b2c3d4e5f6',
    rating: 'incorrect',
    corrected_intent: ['question'],
    corrected_sentiment: 'neutral',
    comment: 'This is Lebanese Arabic, not Gulf'
  })
})

const data = await response.json()
console.log(data.result.acknowledged) // true
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/feedback",
    headers={
        "Authorization": "Bearer nawa_test_sk_xxx",
        "Content-Type": "application/json"
    },
    json={
        "classification_id": "cls_nw_a1b2c3d4e5f6",
        "rating": "incorrect",
        "corrected_intent": ["question"],
        "corrected_sentiment": "neutral",
        "comment": "This is Lebanese Arabic, not Gulf"
    }
)

data = response.json()
print(data["result"]["acknowledged"])  # True
```

</CodeGroup>

## Response

### Success response (200)

```json
{
  "result": {
    "id": "fb_nw_a1b2c3d4e5f6",
    "object": "feedback",
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "acknowledged": true
  },
  "success": true,
  "errors": [],
  "request_id": "req_nw_a1b2c3d4e5f67890abcdef12"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Feedback ID (`fb_nw_xxx` format) |
| `object` | string | Always `feedback` |
| `classification_id` | string | The classification this feedback applies to |
| `rating` | string | The rating you submitted |
| `acknowledged` | boolean | Always `true` on success |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `classification_id` or `rating`, invalid `rating` value, non-array `corrected_intent`, invalid `corrected_sentiment` |
| 401 | `authentication_error` | Invalid or missing API key |
| 405 | `invalid_request_error` | Non-POST request method |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
