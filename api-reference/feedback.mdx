Submit reinforcement learning from human feedback (RLHF) to continuously improve NAWA's classification accuracy. Reference a previous classification by its `classification_id` and provide the correct values.

<Note>
  This endpoint is **free** -- 0 credits, no cost.
</Note>

<Tip>
  Every feedback submission helps improve accuracy for all NAWA users. We especially value feedback on dialect detection edge cases.
</Tip>

## Request

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `classification_id` | string | Yes | The `id` field (e.g. `cls_nw_xxx`) from a previous classify response. |
| `rating` | string | Yes | Overall rating: `correct`, `incorrect`, `partial`. |
| `corrected_intent` | string[] | No | The correct intent labels (e.g. `["complaint"]`). |
| `corrected_sentiment` | string | No | The correct sentiment: `positive`, `negative`, `neutral`, `mixed`. |
| `comment` | string | No | Free-text explanation of why this correction is needed. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/feedback \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "corrected_intent": ["complaint"],
    "corrected_sentiment": "negative",
    "comment": "This was clearly a complaint, not praise"
  }'
```

```typescript TypeScript
const { data, error } = await nawa.feedback.submit({
  classificationId: 'cls_nw_a1b2c3d4e5f6',
  rating: 'incorrect',
  correctedIntent: ['complaint'],
  correctedSentiment: 'negative',
  comment: 'This was clearly a complaint, not praise'
})
```

```python Python
result = nawa.feedback.submit(
    classification_id="cls_nw_a1b2c3d4e5f6",
    rating="incorrect",
    corrected_intent=["complaint"],
    corrected_sentiment="negative",
    comment="This was clearly a complaint, not praise"
)
```

</CodeGroup>

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "id": "fb_nw_a1b2c3d4e5f6",
    "object": "feedback",
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "acknowledged": true
  },
  "errors": [],
  "request_id": "req_nw_fb1a2b3c4d5e"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Feedback ID in `fb_nw_xxx` format |
| `object` | string | Always `"feedback"` |
| `classification_id` | string | The classification this feedback refers to |
| `rating` | string | The rating you submitted |
| `acknowledged` | boolean | Always `true` on success |
