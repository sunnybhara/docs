Submit feedback on a classification result to improve NAWA's accuracy through reinforcement learning from human feedback (RLHF). This endpoint is **free** for all tiers and uses quorum-based pattern confirmation.

<Tip>
  Every feedback submission helps improve accuracy for all NAWA users. Feedback on dialect detection edge cases is especially valuable.
</Tip>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `classification_id` | string | Yes | The `cls_nw_xxx` ID from the original classification response. |
| `rating` | string | Yes | Overall rating: `correct`, `incorrect`, `partial`. |
| `corrected_intent` | string[] | No | The correct intent labels (e.g. `["question", "complaint"]`). |
| `corrected_sentiment` | string | No | The correct sentiment: `positive`, `negative`, `neutral`, `mixed`. |
| `comment` | string | No | Free-text explanation of why this correction is needed. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/feedback \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "corrected_intent": ["complaint"],
    "corrected_sentiment": "negative",
    "comment": "This is Lebanese Arabic, not Gulf. The user is complaining."
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const result = await nawa.feedback.submit({
  classificationId: 'cls_nw_a1b2c3d4e5f6',
  rating: 'incorrect',
  correctedIntent: ['complaint'],
  correctedSentiment: 'negative',
  comment: 'This is Lebanese Arabic, not Gulf. The user is complaining.'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.feedback.submit(
    classification_id="cls_nw_a1b2c3d4e5f6",
    rating="incorrect",
    corrected_intent=["complaint"],
    corrected_sentiment="negative",
    comment="This is Lebanese Arabic, not Gulf. The user is complaining."
)
```

</CodeGroup>

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "id": "fb_nw_a1b2c3d4e5f6",
    "object": "feedback",
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "acknowledged": true
  },
  "errors": [],
  "request_id": "req_nw_fb_abc123def456"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Feedback ID (`fb_nw_xxx` format) |
| `object` | string | Always `feedback` |
| `classification_id` | string | The classification this feedback applies to |
| `rating` | string | The submitted rating |
| `acknowledged` | boolean | Always `true` on success |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `classification_id` or `rating`. Invalid rating value. |
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
