Submit reinforcement learning from human feedback (RLHF) on classification results. This endpoint is **free** for all tiers and helps improve NAWA's accuracy through quorum-based pattern confirmation.

<Tip>
  Every feedback submission helps improve accuracy for all NAWA users. When two or more users submit the same correction, the pattern is confirmed and applied to future classifications.
</Tip>

## Request

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `classification_id` | string | Yes | The `id` field (`cls_nw_xxx`) from the original classification response. |
| `rating` | string | Yes | Your assessment: `correct`, `incorrect`, or `partial`. |
| `corrected_intent` | string[] | No | The correct intent(s), if the classification was wrong. |
| `corrected_sentiment` | string | No | The correct sentiment: `positive`, `negative`, `neutral`, `mixed`. |
| `comment` | string | No | Free-text explanation of why this correction is needed. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/feedback \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "corrected_intent": ["complaint"],
    "corrected_sentiment": "negative",
    "comment": "This is a complaint about audio quality, not praise"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.feedback.submit({
  classificationId: 'cls_nw_a1b2c3d4e5f6',
  rating: 'incorrect',
  correctedIntent: ['complaint'],
  correctedSentiment: 'negative',
  comment: 'This is a complaint about audio quality, not praise'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.feedback.submit(
    classification_id="cls_nw_a1b2c3d4e5f6",
    rating="incorrect",
    corrected_intent=["complaint"],
    corrected_sentiment="negative",
    comment="This is a complaint about audio quality, not praise",
)
```

</CodeGroup>

## Response

### Success response (200)

```json
{
  "result": {
    "id": "fb_nw_a1b2c3d4e5f6",
    "object": "feedback",
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "acknowledged": true
  },
  "success": true,
  "errors": [],
  "request_id": "req_nw_a1b2c3d4e5f67890abcdef12"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Feedback ID (`fb_nw_` prefix) |
| `object` | string | Always `feedback` |
| `classification_id` | string | The classification this feedback refers to |
| `rating` | string | The rating you submitted |
| `acknowledged` | boolean | Always `true` on success |
