Submit reinforcement learning from human feedback (RLHF) to continuously improve NAWA's classification accuracy. Uses quorum-based pattern confirmation for model improvement.

<Note>
  This endpoint is **free** -- 0 credits, no cost.
</Note>

<Tip>
  Every feedback submission helps improve accuracy for all NAWA users. We especially value feedback on dialect detection edge cases.
</Tip>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `classification_id` | string | Yes | The `id` from a classification response (format: `cls_nw_xxx`). |
| `rating` | string | Yes | Your assessment: `correct`, `incorrect`, or `partial`. |
| `corrected_intent` | string[] | No | Array of correct intent labels if the classification was wrong. |
| `corrected_sentiment` | string | No | Correct sentiment: `positive`, `negative`, `neutral`, `mixed`. |
| `comment` | string | No | Free-text explanation of why this correction is needed. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/feedback \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "corrected_sentiment": "negative",
    "corrected_intent": ["complaint"],
    "comment": "This is Lebanese Arabic, not Gulf. The sentiment is negative."
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.feedback.submit({
  classificationId: 'cls_nw_a1b2c3d4e5f6',
  rating: 'incorrect',
  correctedSentiment: 'negative',
  correctedIntent: ['complaint'],
  comment: 'This is Lebanese Arabic, not Gulf. The sentiment is negative.'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.feedback.submit(
    classification_id="cls_nw_a1b2c3d4e5f6",
    rating="incorrect",
    corrected_sentiment="negative",
    corrected_intent=["complaint"],
    comment="This is Lebanese Arabic, not Gulf. The sentiment is negative."
)
```

</CodeGroup>

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "id": "fb_nw_x1y2z3",
    "object": "feedback",
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "acknowledged": true
  },
  "errors": [],
  "request_id": "req_nw_fb_abc123"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique feedback ID (format: `fb_nw_xxx`) |
| `object` | string | Always `feedback` |
| `classification_id` | string | The classification this feedback refers to |
| `rating` | string | The rating you submitted |
| `acknowledged` | boolean | Always `true` on success |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `classification_id` or `rating`, invalid `rating` value, invalid `corrected_sentiment`, `corrected_intent` not an array |
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
