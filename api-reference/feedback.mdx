Submit reinforcement learning from human feedback (RLHF) to continuously improve NAWA classification accuracy. This endpoint is **free** for all tiers.

<Tip>
  Every feedback submission helps improve accuracy for all NAWA users. Feedback is processed through a quorum-based pattern confirmation system.
</Tip>

## Request

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `classification_id` | string | Yes | The `id` field (`cls_nw_xxx`) from a classify response. |
| `rating` | string | Yes | Overall rating: `correct`, `incorrect`, or `partial`. |
| `corrected_intent` | string[] | No | Array of correct intents if the classification was wrong, e.g. `["question", "complaint"]`. |
| `corrected_sentiment` | string | No | Correct sentiment: `positive`, `negative`, `neutral`, or `mixed`. |
| `comment` | string | No | Free-text explanation of why this correction is needed. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/feedback \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "corrected_intent": ["complaint"],
    "corrected_sentiment": "negative",
    "comment": "This was a complaint about audio quality, not a question"
  }'
```

```typescript TypeScript
const { data, error } = await nawa.feedback.submit({
  classificationId: 'cls_nw_a1b2c3d4e5f6',
  rating: 'incorrect',
  correctedIntent: ['complaint'],
  correctedSentiment: 'negative',
  comment: 'This was a complaint about audio quality, not a question'
})
```

```python Python
result = nawa.feedback.submit(
    classification_id="cls_nw_a1b2c3d4e5f6",
    rating="incorrect",
    corrected_intent=["complaint"],
    corrected_sentiment="negative",
    comment="This was a complaint about audio quality, not a question"
)
```

</CodeGroup>

## Response

### Success response (200)

```json
{
  "result": {
    "id": "fb_nw_a1b2c3d4e5f6",
    "object": "feedback",
    "classification_id": "cls_nw_a1b2c3d4e5f6",
    "rating": "incorrect",
    "acknowledged": true
  },
  "success": true,
  "errors": [],
  "request_id": "req_nw_fb_abc123def456ghijkl78"
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Feedback ID (`fb_nw_xxx` format) |
| `object` | string | Always `"feedback"` |
| `classification_id` | string | The classification this feedback refers to |
| `rating` | string | The rating you submitted |
| `acknowledged` | boolean | Always `true` on success |
