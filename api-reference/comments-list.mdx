Retrieve your classified comments with filtering and offset-based pagination. This endpoint is **free**.

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `platform` | string | No | Filter by platform: `youtube`, `instagram`, `twitter`, or `facebook`. |
| `intent` | string | No | Filter by intent: `question`, `complaint`, `praise`, `suggestion`, `spam`, or `other`. |
| `sentiment` | string | No | Filter by sentiment: `positive`, `negative`, `neutral`, or `mixed`. |
| `dialect` | string | No | Filter by dialect: `gulf`, `egyptian`, `levantine`, or `msa`. |
| `limit` | integer | No | Results per page (1 to 100). Default: `25`. |
| `offset` | integer | No | Number of results to skip. Default: `0`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/comments?platform=youtube&intent=question&limit=10&offset=0" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const result = await nawa.comments.list({
  platform: 'youtube',
  intent: 'question',
  limit: 10,
  offset: 0
})

console.log(result.comments)
console.log(result.pagination.has_more)
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="nawa_live_sk_xxx")

result = nawa.comments.list(
    platform="youtube",
    intent="question",
    limit=10,
    offset=0,
)

print(result.comments)
print(result.pagination.has_more)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `comments` | array | List of classified comment objects |
| `comments[].id` | string | Classification ID |
| `comments[].request_id` | string | Original request ID |
| `comments[].text` | string | The classified text |
| `comments[].platform` | string | Source platform |
| `comments[].intent` | string | Detected intent |
| `comments[].sentiment` | string | Detected sentiment |
| `comments[].dialect` | string | Detected Arabic dialect |
| `comments[].created_at` | string | ISO 8601 timestamp |
| `pagination.total` | integer | Total matching comments |
| `pagination.limit` | integer | Results per page |
| `pagination.offset` | integer | Current offset |
| `pagination.has_more` | boolean | Whether more results exist |

### Paginating through results

Use `offset` to page through results:

```bash
# Page 1
curl "https://api.trynawa.com/v1/comments?limit=25&offset=0" \
  -H "Authorization: Bearer nawa_live_sk_xxx"

# Page 2
curl "https://api.trynawa.com/v1/comments?limit=25&offset=25" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

### Error responses

| Status | Type | When |
|--------|------|------|
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
