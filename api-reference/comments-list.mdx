Retrieve classified comments with filtering and pagination. Results are scoped to the calling API key. This endpoint is **free**.

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `platform` | string | No | Filter by platform: `youtube`, `instagram`, `twitter`, or `facebook`. |
| `intent` | string | No | Filter by intent: `question`, `complaint`, `praise`, `suggestion`, `spam`, or `other`. |
| `sentiment` | string | No | Filter by sentiment: `positive`, `negative`, `neutral`, or `mixed`. |
| `dialect` | string | No | Filter by dialect: `gulf`, `egyptian`, `levantine`, or `msa`. |
| `limit` | integer | No | Results per page (1-100). Default: `25`. |
| `offset` | integer | No | Number of results to skip. Default: `0`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/comments?platform=youtube&intent=question&limit=10" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const params = new URLSearchParams({
  platform: 'youtube',
  intent: 'question',
  limit: '10'
})

const response = await fetch(`https://api.trynawa.com/v1/comments?${params}`, {
  headers: {
    'Authorization': 'Bearer nawa_test_sk_xxx'
  }
})

const data = await response.json()
for (const comment of data.result.comments) {
  console.log(comment.text, comment.intent)
}
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/comments",
    headers={"Authorization": "Bearer nawa_test_sk_xxx"},
    params={
        "platform": "youtube",
        "intent": "question",
        "limit": 10
    }
)

data = response.json()
for comment in data["result"]["comments"]:
    print(comment["text"], comment["intent"])
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `comments` | object[] | Array of classified comment objects |
| `comments[].id` | string | Comment record ID |
| `comments[].request_id` | string | Original classification request ID |
| `comments[].text` | string | The classified text |
| `comments[].platform` | string | Source platform |
| `comments[].intent` | string | Detected intent |
| `comments[].sentiment` | string | Detected sentiment |
| `comments[].dialect` | string | Detected dialect |
| `comments[].created_at` | string | ISO 8601 timestamp |
| `pagination.total` | integer | Total matching comments |
| `pagination.limit` | integer | Page size used |
| `pagination.offset` | integer | Current offset |
| `pagination.has_more` | boolean | Whether more results exist |

<Note>
  If comment history storage is not yet enabled for your account, the endpoint returns an empty array with a `message` field explaining the status.
</Note>

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Invalid filter values |
| 401 | `authentication_error` | Invalid or missing API key |
| 405 | `invalid_request_error` | Non-GET request method |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
