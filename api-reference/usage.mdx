Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `endpoint`, `day`. Default: none (aggregate only). |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2026-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer $NAWA_API_KEY"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.usage({
  from: '2026-01-01T00:00:00Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.usage(
    from_date="2026-01-01T00:00:00Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total credit cost in USD |
| `usage.cache_hits` | integer | Number of semantic cache hits |
| `usage.cache_hit_rate` | number | Ratio of cache hits to total requests (0-1) |
| `usage.period` | object | Start and end dates for the query |
| `by_endpoint` | array | Breakdown by endpoint (when `group_by=endpoint`) |
| `by_endpoint[].endpoint` | string | Endpoint path |
| `by_endpoint[].requests` | integer | Request count for this endpoint |
| `by_endpoint[].cost` | number | Total cost for this endpoint in USD |
| `by_endpoint[].cache_hits` | integer | Cache hits for this endpoint |
| `by_endpoint[].avg_latency_ms` | number | Average latency in milliseconds |
| `by_day` | array | Breakdown by day (when `group_by=day`) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
