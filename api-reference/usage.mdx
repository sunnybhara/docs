Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `endpoint` or `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2025-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```typescript TypeScript
const response = await fetch(
  "https://api.trynawa.com/v1/usage?from=2025-01-01T00:00:00Z&group_by=endpoint",
  {
    headers: { Authorization: "Bearer nawa_live_sk_xxx" },
  },
)

const data = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/usage",
    headers={"Authorization": "Bearer nawa_live_sk_xxx"},
    params={
        "from": "2025-01-01T00:00:00Z",
        "group_by": "endpoint",
    },
)

data = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total cost in USD |
| `usage.cache_hits` | integer | Number of cache hits |
| `usage.cache_hit_rate` | number | Cache hit ratio (0--1) |
| `usage.period` | object | The date range queried (`from` and `to`, null if not specified) |
| `by_endpoint` | array | Breakdown per endpoint (when `group_by=endpoint`) |
| `by_day` | array | Breakdown per day (when `group_by=day`) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Invalid `group_by` value (must be `endpoint` or `day`) |
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
