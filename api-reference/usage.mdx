Retrieve API usage statistics including request counts, credit consumption, and cache hit rates.

<Note>
  This endpoint is **free** -- 0 credits, no cost.
</Note>

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group results by: `endpoint`, `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2025-01-01T00:00:00Z&to=2025-01-31T23:59:59Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const { data, error } = await nawa.usage.get({
  from: '2025-01-01T00:00:00Z',
  to: '2025-01-31T23:59:59Z',
  groupBy: 'endpoint'
})
```

```python Python
result = nawa.usage.get(
    from_date="2025-01-01T00:00:00Z",
    to_date="2025-01-31T23:59:59Z",
    group_by="endpoint",
)
```

</CodeGroup>

## Response

```json
{
  "success": true,
  "result": {
    "usage": {
      "total_requests": 12450,
      "total_cost": 68.70,
      "cache_hits": 2231,
      "cache_hit_rate": 0.18,
      "period": {
        "from": "2025-01-01T00:00:00Z",
        "to": "2025-01-31T23:59:59Z"
      }
    },
    "by_endpoint": [
      {
        "endpoint": "/v1/classify",
        "requests": 8500,
        "cost": 51.00,
        "cache_hits": 1955
      },
      {
        "endpoint": "/v1/translate",
        "requests": 1200,

        "cache_hits": 276
      }
    ]
  },
  "errors": [],
  "request_id": "req_nw_usg_abc123"

```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total cost in USD |
| `usage.cache_hits` | integer | Number of semantic cache hits |
| `usage.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `usage.period` | object | Time range with `from` and `to` (ISO 8601) |
| `by_endpoint` | object[] | Breakdown by endpoint with `endpoint`, `requests`, `cost`, `cache_hits` |
