Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: all time. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `endpoint` or `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2026-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/usage?group_by=endpoint', {
  headers: { Authorization: `Bearer ${process.env.NAWA_API_KEY}` }
})
const data = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/usage",
    params={"group_by": "endpoint"},
    headers={"Authorization": "Bearer your_api_key"}
)
data = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total credit cost in USD |
| `usage.cache_hits` | integer | Number of requests served from cache |
| `usage.cache_hit_rate` | number | Ratio of cache hits to total requests (0--1) |
| `usage.period` | object | The time period queried (`from` and `to`) |
| `by_endpoint` | array | Per-endpoint breakdown with `endpoint`, `requests`, `cost`, `cache_hits` |
| `by_day` | array | Per-day breakdown with `date`, `requests`, `cost`, `cache_hits` (when `group_by=day`) |
