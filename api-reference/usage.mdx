Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `day`, `endpoint`. Default: `day`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?group_by=endpoint" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="nawa_live_sk_xxx")

result = nawa.usage.get(group_by="endpoint")

print(result.usage.total_requests)  # 12450
print(result.usage.total_cost)      # 68.70
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.usage.get({
  groupBy: 'endpoint'
})

console.log(data.usage.totalRequests)  // 12450
console.log(data.usage.totalCost)      // 68.70
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.period.from` | string | Start of the reporting period (ISO 8601) |
| `usage.period.to` | string | End of the reporting period (ISO 8601) |
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total credit cost in USD |
| `usage.cache_hits` | integer | Total semantic cache hits |
| `usage.cache_hit_rate` | number | Cache hit rate (0--1) |
| `by_endpoint[].endpoint` | string | API endpoint path |
| `by_endpoint[].requests` | integer | Request count for this endpoint |
| `by_endpoint[].cost` | number | Cost in USD for this endpoint |
| `by_endpoint[].cache_hits` | integer | Cache hits for this endpoint |
