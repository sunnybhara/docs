Retrieve your API usage statistics including request counts, cost breakdown, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `endpoint` or `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2025-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const result = await nawa.usage({
  from: '2025-01-01T00:00:00Z',
  groupBy: 'endpoint'
})

console.log(result.usage.total_requests) // 12450
console.log(result.usage.total_cost)     // 68.70
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="nawa_live_sk_xxx")

result = nawa.usage(
    from_date="2025-01-01T00:00:00Z",
    group_by="endpoint"
)

print(result.usage.total_requests)  # 12450
print(result.usage.total_cost)      # 68.70
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total cost in USD |
| `usage.cache_hits` | integer | Number of cache hits |
| `usage.cache_hit_rate` | number | Cache hit rate (0-1) |
| `usage.period` | object | Date range (`from`, `to`) |
| `by_endpoint[].endpoint` | string | Endpoint path |
| `by_endpoint[].requests` | integer | Request count |
| `by_endpoint[].cost` | number | Cost in USD |
| `by_endpoint[].cache_hits` | integer | Cache hit count |
| `by_day[].date` | string | Date (`YYYY-MM-DD`) |
| `by_day[].requests` | integer | Request count |
| `by_day[].cost` | number | Cost in USD |
| `by_day[].cache_hits` | integer | Cache hit count |

### Error responses

| Status | Type | When |
|--------|------|------|
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
