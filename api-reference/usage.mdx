Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `endpoint` or `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2025-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const response = await fetch(
  'https://api.trynawa.com/v1/usage?group_by=endpoint',
  {
    headers: {
      'Authorization': 'Bearer nawa_test_sk_xxx'
    }
  }
)

const data = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/usage",
    headers={"Authorization": "Bearer nawa_test_sk_xxx"},
    params={"group_by": "endpoint"}
)

data = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total cost in USD |
| `usage.cache_hits` | integer | Number of cache hits |
| `usage.cache_hit_rate` | number | Cache hit rate (0-1) |
| `usage.period` | object | Time period with `from` and `to` |
| `by_endpoint` | array | Breakdown by endpoint (when `group_by=endpoint`) |
| `by_endpoint[].endpoint` | string | Endpoint path |
| `by_endpoint[].requests` | integer | Request count |
| `by_endpoint[].cost` | number | Total cost in USD |
| `by_endpoint[].cache_hits` | integer | Cache hits for this endpoint |
| `by_day` | array | Breakdown by day (when `group_by=day`) |
| `by_day[].date` | string | Date (YYYY-MM-DD) |
| `by_day[].requests` | integer | Request count |
| `by_day[].cost` | number | Total cost in USD |
| `by_day[].cache_hits` | integer | Cache hits for this day |

### Error responses

| Status | Type | When |
|--------|------|------|
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
