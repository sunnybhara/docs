Retrieve your API usage statistics including request counts, total cost, and cache hit rates. This endpoint is **free**.

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601), e.g. `2025-01-01T00:00:00Z` |
| `to` | string | No | End date (ISO 8601), e.g. `2025-01-31T23:59:59Z` |
| `group_by` | string | No | Group by: `endpoint` or `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2025-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.usage({
  from: '2025-01-01T00:00:00Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.usage(
    from_date="2025-01-01T00:00:00Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Group by `day` example

```bash
curl "https://api.trynawa.com/v1/usage?group_by=day&from=2025-01-01" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

Returns `by_day` with date-keyed entries:

```json
{
  "by_day": [
    {"date": "2025-01-15", "requests": 450, "cost": 2.7, "cache_hits": 103},
    {"date": "2025-01-14", "requests": 420, "cost": 2.52, "cache_hits": 97}
  ]
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total cost in USD |
| `usage.cache_hits` | integer | Number of cache hits |
| `usage.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `usage.period` | object | The `from` and `to` dates of the query |
| `by_endpoint[].endpoint` | string | Endpoint path |
| `by_endpoint[].requests` | integer | Request count |
| `by_endpoint[].cost` | number | Total cost for this endpoint |
| `by_endpoint[].cache_hits` | integer | Cache hits for this endpoint |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Invalid `group_by` value (must be `endpoint` or `day`) |
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
