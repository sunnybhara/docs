Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `endpoint` or `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2025-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.usage({
  from: '2025-01-01T00:00:00Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.usage(
    from_date="2025-01-01T00:00:00Z",
    group_by="endpoint"
)
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total cost in USD |
| `usage.cache_hits` | integer | Total cache hits |
| `usage.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `usage.period` | object | The query time range |
| `by_endpoint` / `by_day` | array | Breakdown grouped by the `group_by` parameter |

When grouped by `endpoint`, each item contains:

| Field | Type | Description |
|-------|------|-------------|
| `endpoint` | string | The API endpoint path |
| `requests` | integer | Number of requests |
| `cost` | number | Total cost in USD |
| `cache_hits` | integer | Number of cache hits |

When grouped by `day`, each item contains:

| Field | Type | Description |
|-------|------|-------------|
| `date` | string | Date in `YYYY-MM-DD` format |
| `requests` | integer | Number of requests |
| `cost` | number | Total cost in USD |
| `cache_hits` | integer | Number of cache hits |
