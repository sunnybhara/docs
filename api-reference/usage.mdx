Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601), e.g. `2026-01-01T00:00:00Z` |
| `to` | string | No | End date (ISO 8601), e.g. `2026-01-31T23:59:59Z` |
| `group_by` | string | No | Group by: `endpoint`, `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2026-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const response = await fetch(
  'https://api.trynawa.com/v1/usage?group_by=endpoint',
  { headers: { 'Authorization': `Bearer ${process.env.NAWA_API_KEY}` } }
)
const { result } = await response.json()
console.log(result.usage.total_requests)
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/usage",
    params={"group_by": "endpoint"},
    headers={"Authorization": "Bearer your_api_key"}
)
data = response.json()
print(data["result"]["usage"]["total_requests"])
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total credit cost in USD |
| `usage.cache_hits` | integer | Number of requests served from cache |
| `usage.cache_hit_rate` | number | Ratio of cache hits to total requests (0-1) |
| `usage.period` | object | The queried time range (`from` and `to`, null if not specified) |
| `by_endpoint` / `by_day` | array | Breakdown grouped by the `group_by` parameter |

<Info>
  When grouped by `day`, each item has a `date` field (YYYY-MM-DD format) instead of `endpoint`.
</Info>

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Invalid `group_by` value (must be `endpoint` or `day`) |
| 401 | `authentication_error` | Invalid or missing API key |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal error |
