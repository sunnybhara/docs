Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group results by: `day`, `week`, `month`, or `endpoint`. Default: `day`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2026-03-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const response = await fetch('https://api.trynawa.com/v1/usage?group_by=endpoint', {
  headers: { Authorization: `Bearer ${process.env.NAWA_API_KEY}` }
})
const usage = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/usage",
    params={"group_by": "endpoint"},
    headers={"Authorization": "Bearer your_api_key"}
)
usage = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `total_requests` | integer | Total API requests in the period |
| `total_cost` | number | Total credit cost in USD |
| `cache_hit_rate` | number | Proportion of requests served from cache (0-1) |
| `by_endpoint` | array | Breakdown by endpoint with request count, cost, cache hits, and average latency |
| `balance` | object | Current balance, total credits purchased, and total credits used |
