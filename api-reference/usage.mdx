Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: all time. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `endpoint`, `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2026-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
const params = new URLSearchParams({
  from: "2026-01-01T00:00:00Z",
  group_by: "endpoint",
});

const response = await fetch(
  `https://api.trynawa.com/v1/usage?${params}`,
  {
    headers: {
      "Authorization": "Bearer nawa_test_sk_xxx",
    },
  },
);

const data = await response.json();
console.log(data.result.usage);
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/usage",
    headers={"Authorization": "Bearer nawa_test_sk_xxx"},
    params={
        "from": "2026-01-01T00:00:00Z",
        "group_by": "endpoint",
    },
)

data = response.json()
print(data["result"]["usage"])
```

</CodeGroup>

### Usage fields

| Field | Type | Description |
|-------|------|-------------|
| `total_requests` | integer | Total API requests in the period |
| `total_cost` | number | Total credit cost in USD |
| `cache_hits` | integer | Number of requests served from cache |
| `cache_hit_rate` | number | Percentage of requests served from cache (0-1) |
| `period` | object | The `from` and `to` timestamps for the query |

### Breakdown fields

The `by_{group_by}` array contains per-group breakdowns sorted by request count (descending):

When grouped by `endpoint`:

| Field | Type | Description |
|-------|------|-------------|
| `endpoint` | string | API endpoint path |
| `requests` | integer | Total requests |
| `cost` | number | Total cost in USD |
| `cache_hits` | integer | Number of cache hits |

When grouped by `day`:

| Field | Type | Description |
|-------|------|-------------|
| `date` | string | Date in `YYYY-MM-DD` format |
| `requests` | integer | Total requests |
| `cost` | number | Total cost in USD |
| `cache_hits` | integer | Number of cache hits |
