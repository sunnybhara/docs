Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). Default: start of current month. |
| `to` | string | No | End date (ISO 8601). Default: now. |
| `group_by` | string | No | Group by: `endpoint` or `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2026-03-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```typescript TypeScript
const response = await fetch(
  "https://api.trynawa.com/v1/usage?from=2026-03-01T00:00:00Z&group_by=endpoint",
  {
    headers: { Authorization: "Bearer nawa_live_sk_xxx" },
  }
);

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/usage",
    headers={"Authorization": "Bearer nawa_live_sk_xxx"},
    params={"from": "2026-03-01T00:00:00Z", "group_by": "endpoint"},
)

data = response.json()
```

</CodeGroup>

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total cost in USD |
| `usage.cache_hits` | integer | Total cache hits |
| `usage.cache_hit_rate` | number | Cache hit ratio (0-1) |
| `usage.period` | object | The queried time period with `from` and `to` |
| `by_endpoint` | array | Per-endpoint breakdown with `endpoint`, `requests`, `cost`, `cache_hits` |
| `by_day` | array | Per-day breakdown (when `group_by=day`) with `date`, `requests`, `cost`, `cache_hits` |
