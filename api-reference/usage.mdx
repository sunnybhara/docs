Retrieve your API usage statistics including request counts, credit consumption, and cache hit rates. This endpoint is **free**.

## Request

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `from` | string | No | Start date (ISO 8601). |
| `to` | string | No | End date (ISO 8601). |
| `group_by` | string | No | Group by: `endpoint` or `day`. Default: `endpoint`. |

### Example request

<CodeGroup>

```bash cURL
curl "https://api.trynawa.com/v1/usage?from=2026-01-01T00:00:00Z&group_by=endpoint" \
  -H "Authorization: Bearer nawa_test_sk_xxx"
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.usage({
  from: '2026-01-01T00:00:00Z',
  groupBy: 'endpoint'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.usage(
    from_date="2026-01-01T00:00:00Z",
    group_by="endpoint"
)
```

</CodeGroup>

## Response

### Success response (200)

```json
{
  "success": true,
  "result": {
    "usage": {
      "total_requests": 12450,
      "total_cost": 68.7,
      "cache_hits": 2835,
      "cache_hit_rate": 0.23,
      "period": {
        "from": "2026-01-01T00:00:00Z",
        "to": null
      }
    },
    "by_endpoint": [
      {
        "endpoint": "/v1/classify",
        "requests": 8500,
        "cost": 51.0,
        "cache_hits": 1955
      },
      {
        "endpoint": "/v1/rubric-classify",
        "requests": 2200,
        "cost": 6.6,
        "cache_hits": 880
      },
      {
        "endpoint": "/v1/comments/reply",
        "requests": 1750,
        "cost": 14.0,
        "cache_hits": 0
      }
    ]
  },
  "errors": [],
  "request_id": "req_nw_usg_abc123def456ghij78"

```

When `group_by=day`, the breakdown key changes to `by_day` with a `date` field:

```json
{
  "by_day": [
    {
      "date": "2026-01-15",
      "requests": 450,
      "cost": 2.7,
      "cache_hits": 103
    }
  ]
}
```

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `usage.total_requests` | integer | Total API requests in the period |
| `usage.total_cost` | number | Total credit cost in USD |
| `usage.cache_hits` | integer | Number of requests served from cache |
| `usage.cache_hit_rate` | number | Ratio of cache hits to total requests (0-1) |
| `usage.period` | object | The time range queried (`from` and `to`) |
| `by_endpoint` / `by_day` | array | Breakdown grouped by the `group_by` parameter |
