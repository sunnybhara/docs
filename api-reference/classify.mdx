Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction powered by NAWA's multi-provider AI pipeline.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. |
| `context` | object | No | Optional context object to improve classification accuracy. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟"
  }'
```

```typescript TypeScript
const response = await fetch("https://api.trynawa.com/v1/classify", {
  method: "POST",
  headers: {
    "Authorization": "Bearer nawa_test_sk_xxx",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    text: "متى الجزء الثاني؟",
  }),
});

const data = await response.json();
console.log(data.result);
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/classify",
    headers={
        "Authorization": "Bearer nawa_test_sk_xxx",
        "Content-Type": "application/json",
    },
    json={
        "text": "متى الجزء الثاني؟",
    },
)

data = response.json()
print(data["result"])
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging (`req_nw_xxx` format) |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Provider` | AI provider used for this request (`claude`, `gemini`, or `allam`) |
| `X-NAWA-Latency` | Processing time in milliseconds |
| `X-NAWA-Calibration-Version` | Calibration version applied to the result |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_nw_xxx` format) |
| `object` | string | Always `"classification"` |
| `intent` | string[] | Detected intents: `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language code: `ar`, `en`, `mixed` |
| `dialect` | string or null | Arabic dialect if detected: `gulf`, `egyptian`, `levantine`, `msa`. Null for non-Arabic text. |
| `dialect_confidence` | number or null | Dialect detection confidence (0-1). Null for non-Arabic text. |
| `requires_response` | boolean | Whether the comment warrants a reply |
| `priority` | string | Priority level: `high`, `medium`, `low` |
| `suggested_reply` | object | Suggested reply with `text` (string) and `direction` (`ltr` or `rtl`) |
| `provider` | string | AI provider used: `claude`, `gemini`, `allam` |
| `model` | string | Model identifier used for classification |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | integer or null | Token count (null if not tracked) |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits deducted |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or invalid parameters |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
