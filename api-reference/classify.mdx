Classify an Arabic or English comment with a single request. Returns intent, sentiment, dialect detection, priority scoring, and a suggested reply powered by HUMAIN's ALLaM model with Claude fallback.

## Request

<Note>
  Cost: **6 credits** ($0.006) per request. Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |
| `X-Request-Id` | No | Optional caller-supplied request ID. Auto-generated as `req_nw_xxx` if absent. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify. Must be non-empty. |
| `context` | object | No | Additional context object to improve classification accuracy. |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, or `allam`. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'متى الجزء الثاني؟'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(text="متى الجزء الثاني؟")
```

</CodeGroup>

```json
  "success": true,

    "dialect": "gulf",

    "requires_response": true,

    "suggested_reply": {
      "text": "إن شاء الله الجزء الثاني قريب!",
      "direction": "rtl"
    },
    "provider": "claude",
    "model": "claude-haiku-4-5-20251001",
    "fallback_used": false,
    "tokens_used": null,
    "cost_usd": 0.006,
    "credits_used": 6
  },
  "errors": [],

```

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-NAWA-Environment` | `sandbox` or `live` |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost) |
| `X-NAWA-Provider` | AI provider used (e.g., `claude`, `allam`, `gemini`) |
| `X-NAWA-Latency` | Request latency in milliseconds |
| `X-NAWA-Calibration-Version` | Calibration version applied to results |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID in `cls_nw_xxx` format |
| `object` | string | Always `"classification"` |
| `intent` | string[] | Detected intents: `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language code (`ar`, `en`, `mixed`) |
| `dialect` | string or null | `gulf`, `egyptian`, `levantine`, `msa`. Null if not Arabic. |
| `dialect_confidence` | number or null | Confidence score (0-1). Null if not Arabic. |
| `requires_response` | boolean | Whether this comment warrants a reply |
| `priority` | string | `high`, `medium`, or `low` |
| `suggested_reply` | object | Suggested reply with `text` and `direction` (`rtl` or `ltr`) |
| `provider` | string | AI provider used (`claude`, `allam`, `gemini`) |
| `model` | string | Specific model version used |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | number or null | Token count if available |
| `cost_usd` | number | Cost in USD |
| `credits_used` | number | Credits deducted |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing or empty `text`, invalid `provider` value, invalid JSON body |
| 401 | `authentication_error` | Invalid or missing API key |
| 405 | `invalid_request_error` | Wrong HTTP method (only POST allowed) |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
