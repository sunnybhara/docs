Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction powered by the NAGL language gateway.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Must be non-empty. |
| `context` | object | No | Optional context object to improve classification accuracy. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "ماشاء الله الفيديو حلو مرة"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const result = await nawa.classify({
  text: 'ماشاء الله الفيديو حلو مرة'
})

console.log(result.data.intent)      // ["praise"]
console.log(result.data.sentiment)   // "positive"
console.log(result.data.dialect)     // "gulf"
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(text="ماشاء الله الفيديو حلو مرة")

print(result.data.intent)      # ["praise"]
print(result.data.sentiment)   # "positive"
print(result.data.dialect)     # "gulf"
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-NAWA-Provider` | AI provider used: `claude`, `gemini`, or `allam` |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Latency` | Request latency in milliseconds |
| `X-NAWA-Calibration-Version` | Calibration version applied (if any) |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID in `cls_nw_xxx` format |
| `object` | string | Always `"classification"` |
| `intent` | string[] | Detected intents: `praise`, `question`, `complaint`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language: `ar`, `en`, `mixed` |
| `dialect` | string or null | Arabic dialect: `gulf`, `egyptian`, `levantine`, `maghrebi`, `msa`. Null when language is not `ar`. |
| `dialect_confidence` | number or null | Confidence score 0-1. Null when language is not `ar`. |
| `requires_response` | boolean | Whether this comment warrants a response |
| `priority` | string | `high`, `medium`, `low` |
| `suggested_reply` | object | Suggested reply with `text` and `direction` (`rtl` or `ltr`) |
| `provider` | string | AI provider used: `claude`, `gemini`, `allam` |
| `model` | string | Specific model version used |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | integer or null | Token count if available |
| `cost_usd` | number | Cost in USD (always 0.006) |
| `credits_used` | integer | Credits deducted (always 6) |

<Note>
  When `X-NAWA-Cache: HIT`, the cached result is returned and no credits are deducted. Calibration thresholds are still applied after cache retrieval.
</Note>

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing or empty `text` |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
