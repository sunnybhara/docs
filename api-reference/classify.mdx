Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction powered by HUMAIN's ALLaM model.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Must be non-empty. |
| `context` | object | No | Optional context object for classification. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "ماشاء الله الفيديو حلو مرة"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'ماشاء الله الفيديو حلو مرة'
})

if (data) {
  console.log(data.result.intent)     // ["praise"]
  console.log(data.result.sentiment)  // "positive"
  console.log(data.result.dialect)    // "gulf"
}
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(
    text="ماشاء الله الفيديو حلو مرة"
)

print(result.data.intent)     # ["praise"]
print(result.data.sentiment)  # "positive"
print(result.data.dialect)    # "gulf"
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier (`req_nw_` prefix) |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Provider` | AI provider used for this request (`claude`, `gemini`, or `allam`) |
| `X-NAWA-Latency` | Server-side latency in milliseconds |
| `X-NAWA-Calibration-Version` | Calibration version applied to the result |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_nw_` prefix) |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents (e.g., `["praise"]`, `["question", "complaint"]`) |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language: `ar`, `en`, `mixed` |
| `dialect` | string or null | Arabic dialect: `gulf`, `egyptian`, `levantine`, `maghrebi`, `msa`. Null if not Arabic. |
| `dialect_confidence` | number or null | Confidence score (0-1). Null if not Arabic. |
| `requires_response` | boolean | Whether the comment warrants a reply |
| `priority` | string | `high`, `medium`, `low` |
| `suggested_reply` | object | Contains `text` (summary) and `direction` (`ltr` or `rtl`) |
| `provider` | string | AI provider used: `claude`, `gemini`, `allam` |
| `model` | string | Model identifier used for classification |
| `fallback_used` | boolean | Whether the fallback provider was used |
| `tokens_used` | integer or null | Token count (may be null) |
| `cost_usd` | number | Cost in USD for this request |
| `credits_used` | integer | Credits deducted from your balance |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, invalid `provider`, invalid JSON |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
