Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction powered by NAWA's NAGL engine with ALLaM.

<Note>
  Cost: **6 credits ($0.006)** per request. Semantic cache hits are free (`X-NAWA-Cache: HIT`). Free tier keys (sandbox) are not charged credits.
</Note>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, or `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Must be non-empty. |
| `context` | object | No | Optional context object for classification. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_live_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "ماشاء الله الفيديو حلو مرة"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const result = await nawa.classify({
  text: 'ماشاء الله الفيديو حلو مرة'
})

console.log(result.intent)    // ["praise"]
console.log(result.dialect)   // "gulf"
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="nawa_live_sk_xxx")

result = nawa.classify(
    text="ماشاء الله الفيديو حلو مرة"
)

print(result.intent)    # ["praise"]
print(result.dialect)   # "gulf"
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier (`req_nw_xxx` format) |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Provider` | AI provider used (e.g., `claude`, `gemini`, `allam`) |
| `X-NAWA-Latency` | Processing time in milliseconds |
| `X-NAWA-Environment` | `sandbox` or `live` |
| `X-NAWA-Calibration-Version` | Calibration version applied (Enterprise only) |
| `X-RateLimit-Limit` | Maximum requests allowed per minute for your tier |
| `X-RateLimit-Remaining` | Requests remaining in the current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_nw_xxx` format) |
| `object` | string | Always `"classification"` |
| `intent` | string[] | Detected intents, e.g., `["praise"]`, `["question", "complaint"]` |
| `sentiment` | string | `positive`, `negative`, `neutral`, or `mixed` |
| `language` | string | Detected language: `ar`, `en`, or `mixed` |
| `dialect` | string or null | Arabic dialect when `language` is `ar`: `gulf`, `egyptian`, `levantine`, `maghrebi`, or `msa`. Null for non-Arabic text. |
| `dialect_confidence` | number or null | Confidence score (0 to 1) for dialect detection. Null for non-Arabic text. |
| `requires_response` | boolean | Whether the comment requires a response |
| `priority` | string | `high`, `medium`, or `low` |
| `suggested_reply` | object | Contains `text` (suggested reply direction) and `direction` (`rtl` or `ltr`) |
| `provider` | string | AI provider used: `claude`, `gemini`, or `allam` |
| `model` | string | Specific model version used |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | integer or null | Token count (currently null) |
| `cost_usd` | number | Cost in USD for this request |
| `credits_used` | integer | Credits deducted (0 for sandbox keys) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or invalid parameters |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `permission_error` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
