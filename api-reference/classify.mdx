title: "Classify comment"
sidebarTitle: "POST /v1/classify"
description: "Classify an Arabic or English comment by intent, sentiment, dialect, and priority in a single API call."
api: "POST https://api.trynawa.com/v1/classify"
---

Classify any comment with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply. Arabic comments are routed to HUMAIN's ALLaM model for dialect-aware classification. English comments are routed to Claude.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify. Max 5,000 characters. |
| `context` | object | No | Additional context to improve classification accuracy. |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'متى الجزء الثاني؟'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(
    text="متى الجزء الثاني؟"
)
```

</CodeGroup>

    "cost_usd": 0.006,

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Balance` | Current credit balance in USD |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost) |
| `X-NAWA-Provider` | AI provider used for this request |
| `X-NAWA-Calibration-Version` | User-specific calibration version applied |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique classification ID (`cls_nw_xxx` format) |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents: `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language: `ar`, `en`, `mixed` |
| `dialect` | string or null | `gulf`, `egyptian`, `levantine`, `msa`. Returns `null` for English text. |
| `dialect_confidence` | number or null | Confidence score (0-1). Returns `null` for English text. |
| `requires_response` | boolean | Whether the comment needs a reply |
| `priority` | string | Comment priority: `low`, `medium`, `high` |
| `suggested_reply` | object | Contains `text` (reply text) and `direction` (`rtl` or `ltr`) |
| `provider` | string | AI provider used: `allam`, `claude`, `gemini` |
| `model` | string | Model version used for classification |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | number or null | Token count (may be null) |
| `cost_usd` | number | Cost in USD |
| `credits_used` | number | Credits deducted |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, text too long, invalid `provider` |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
