Classify an Arabic comment with a single request. Returns intent, sentiment, dialect, and priority analysis powered by HUMAIN's ALLaM model, with automatic fallback to Claude Haiku 4.5 or Google Gemini.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, or `allam`. If omitted, NAWA selects the optimal provider automatically. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify. Max 5,000 characters. |
| `context` | object | No | Additional context to improve classification accuracy. Arbitrary key-value pairs. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟",
    "context": { "platform": "youtube", "channel": "tech_reviews" }
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'متى الجزء الثاني؟',
  context: { platform: 'youtube', channel: 'tech_reviews' }
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(
    text="متى الجزء الثاني؟",
    context={"platform": "youtube", "channel": "tech_reviews"}
)
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Balance` | Current credit balance in USD |
| `X-NAWA-Balance-Warning` | `low_balance` when below $5 |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost) |
| `X-NAWA-Provider` | AI provider used for this request (`allam`, `claude`, or `gemini`) |
| `X-NAWA-Latency` | Processing time in milliseconds |
| `X-NAWA-Calibration-Version` | Version of calibration thresholds applied |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID in `cls_nw_xxx` format |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents: `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language: `ar`, `en`, or `mixed` |
| `dialect` | string or null | `gulf`, `egyptian`, `levantine`, `msa`. Null if not Arabic. |
| `dialect_confidence` | number or null | Confidence score (0-1). Null if not Arabic. |
| `requires_response` | boolean | Whether this comment warrants a reply |
| `priority` | string | `high`, `medium`, or `low` |
| `suggested_reply` | object | AI-generated reply with `text` and `direction` (`ltr` or `rtl`) |
| `provider` | string | AI provider used: `allam`, `claude`, or `gemini` |
| `model` | string | Model version used for classification |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | number or null | Token count (null when not tracked) |
| `cost_usd` | number | Cost in USD for this request |
| `credits_used` | integer | Credits deducted from your balance |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, text too long, empty string |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
