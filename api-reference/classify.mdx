---

Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply -- powered by HUMAIN's ALLaM model with Claude fallback.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. |
| `context` | object | No | Optional context object for classification. |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST "https://api.trynawa.com/v1/classify" \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'متى الجزء الثاني؟'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(
    text="متى الجزء الثاني؟"
)
```

</CodeGroup>

```json
    "dialect": "gulf",

    "requires_response": true,
    "priority": "medium",
    "suggested_reply": {
      "text": "User is asking about a follow-up",
      "direction": "rtl"
    },
    "provider": "allam",
    "model": "sdaia/allam-1-13b-instruct",
    "fallback_used": false,
    "tokens_used": null,
    "cost_usd": 0.006,
    "credits_used": 6
  },
  "errors": [],
  "request_id": "req_nw_a1b2c3d4e5f67890"

```

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost) |
| `X-NAWA-Provider` | AI provider used (`allam`, `claude`, or `gemini`) |
| `X-NAWA-Latency` | Request latency in milliseconds |
| `X-NAWA-Calibration-Version` | Calibration version applied (Enterprise only) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_nw_xxx` format) |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents: `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | `ar`, `en`, `mixed` |
| `dialect` | string or null | `gulf`, `egyptian`, `levantine`, `msa`. `null` for non-Arabic text. |
| `dialect_confidence` | number or null | Confidence score (0-1). `null` for non-Arabic text. |
| `requires_response` | boolean | Whether the text warrants a response |
| `priority` | string | `high`, `medium`, `low` |
| `suggested_reply` | object | `{text, direction}` -- suggested reply with text direction (`rtl` or `ltr`) |
| `provider` | string | AI provider used (`allam`, `claude`, or `gemini`) |
| `model` | string | Model version used |
| `fallback_used` | boolean | Whether the fallback provider was used |
| `tokens_used` | null | Reserved for future use |
| `cost_usd` | number | Cost of this request in USD |
| `credits_used` | integer | Credits deducted (0 for free keys) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or invalid `provider` |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
