Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and an optional suggested reply powered by the NAGL pipeline.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Override the AI provider: `claude`, `gemini`, `allam`. Used for A/B testing. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Max 5,000 characters. |
| `context` | object | No | Additional context to improve classification accuracy. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_live_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟"
  }'
```

```typescript TypeScript
import { Nawa } from "@nawalabs/sdk";

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY });

const result = await nawa.classify({
  text: "متى الجزء الثاني؟",
});

console.log(result.intent); // ["question"]
console.log(result.suggested_reply?.text);
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="nawa_live_sk_xxx")

result = nawa.classify(text="متى الجزء الثاني؟")

print(result.intent)           # ["question"]
print(result.suggested_reply)
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Provider` | AI provider used: `allam`, `claude`, or `gemini` |
| `X-NAWA-Latency` | Processing latency in milliseconds |
| `X-NAWA-Calibration-Version` | Calibration version applied to this request |
| `X-NAWA-Balance` | Current credit balance in USD |
| `X-RateLimit-Limit` | Rate limit ceiling for the current window |
| `X-RateLimit-Remaining` | Requests remaining in the current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_xxx` format) |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents: `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language: `ar`, `en`, `mixed` |
| `dialect` | string or null | Detected Arabic dialect: `gulf`, `egyptian`, `levantine`, `msa`. Null if not Arabic. |
| `dialect_confidence` | number or null | Confidence score (0-1) for dialect detection. Null if not Arabic. |
| `requires_response` | boolean | Whether the comment warrants a reply |
| `priority` | string | Priority level for response queue |
| `suggested_reply` | object or null | Suggested reply with `text` and `direction` (`rtl` or `ltr`) |
| `provider` | string | AI provider used: `allam`, `claude`, or `gemini` |
| `model` | string | Model version used for classification |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | number or null | Token count (if available) |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits deducted |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or text exceeds 5,000 characters |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
