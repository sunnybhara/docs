Classify Arabic or English text in a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction. Supports semantic caching.

<Note>
  Cost: **$0.002** per request. Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

## Request

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Minimum 1 character. |
| `context` | object | No | Optional context object for classification. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "ماشاء الله الفيديو حلو مرة"
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/classify",
    headers={
        "Authorization": "Bearer nawa_test_sk_xxx",
        "Content-Type": "application/json",
    },
    json={
        "text": "ماشاء الله الفيديو حلو مرة",
    },
)

result = response.json()
print(result["result"]["intent"])     # ["praise"]
print(result["result"]["dialect"])    # "gulf"
```

```typescript TypeScript
const response = await fetch("https://api.trynawa.com/v1/classify", {
  method: "POST",
  headers: {
    Authorization: "Bearer nawa_test_sk_xxx",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    text: "ماشاء الله الفيديو حلو مرة",
  }),
});

const { result } = await response.json();
console.log(result.intent);     // ["praise"]
console.log(result.dialect);    // "gulf"
```

</CodeGroup>

```json
    "cost": 0.002
  },
  "errors": [],

```

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier (`req_nw_xxx` format) |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Provider` | AI provider used for this request |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_nw_xxx` format) |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents (e.g. `["praise"]`, `["question", "complaint"]`) |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | `ar`, `en`, `mixed` |
| `dialect` | string or null | Arabic dialect: `gulf`, `egyptian`, `levantine`, `maghrebi`, `msa`. Null if not Arabic. |
| `dialect_confidence` | number or null | Confidence score (0 to 1). Null if not Arabic. |
| `requires_response` | boolean | Whether the comment warrants a reply |
| `priority` | string | `high`, `medium`, `low` |
| `suggested_reply` | object | Suggested reply with `text` and `direction` (`ltr` or `rtl`) |
| `provider` | string | AI provider used: `claude`, `gemini`, `allam` |
| `model` | string | Model identifier used |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | integer or null | Tokens consumed (null if not tracked) |
| `cost` | number | Cost in USD |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` field |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `permission_error` | Insufficient credits |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
