Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction powered by HUMAIN's ALLaM model. Supports semantic caching.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Min 1 character. |
| `context` | object | No | Optional context object to improve classification accuracy. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'متى الجزء الثاني؟'
})

if (error) {
  console.error(error.code, error.message)
} else {
  console.log(data.intent)     // ["question"]
  console.log(data.dialect)    // "gulf"
  console.log(data.sentiment)  // "neutral"
}
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(
    text="متى الجزء الثاني؟"
)

if result.error:
    print(result.error.code, result.error.message)
else:
    print(result.data.intent)     # ["question"]
    print(result.data.dialect)    # "gulf"
    print(result.data.sentiment)  # "neutral"
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier (`req_nw_xxx` format) |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Provider` | AI provider used: `claude`, `gemini`, or `allam` |
| `X-NAWA-Environment` | `live` or `sandbox` based on your API key |
| `X-NAWA-Calibration-Version` | Calibration version applied to the result |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_nw_xxx` format) |
| `object` | string | Always `"classification"` |
| `intent` | string[] | Detected intents: `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language: `ar`, `en`, `mixed` |
| `dialect` | string or null | Arabic dialect: `gulf`, `egyptian`, `levantine`, `msa`. Null for non-Arabic text. |
| `dialect_confidence` | number or null | Confidence score (0-1) for dialect detection |
| `requires_response` | boolean | Whether the comment warrants a reply |
| `priority` | string | `high`, `medium`, `low` |
| `suggested_reply` | object | Contains `text` (summary) and `direction` (`ltr` or `rtl`) |
| `provider` | string | AI provider used: `claude`, `gemini`, `allam` |
| `model` | string | Specific model version used |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | integer or null | Token count (when available) |
| `cost_usd` | number | Cost in USD for this request |
| `credits_used` | integer | Credits deducted (0 for cache hits) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, invalid `provider` |
| 401 | `authentication_error` | Invalid, expired, or revoked API key |
| 402 | `permission_error` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded or sandbox cap reached |
| 500 | `api_error` | Internal or provider error |
