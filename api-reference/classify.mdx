---

Classify an Arabic comment with a single request. Returns intent, sentiment, dialect, and priority analysis powered by HUMAIN's ALLaM model with automatic Claude fallback.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify. |
| `context` | object | No | Additional context to improve classification accuracy. |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, `allam`. Useful for A/B testing. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'متى الجزء الثاني؟'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(
    text="متى الجزء الثاني؟"
)
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Balance` | Current credit balance in USD |
| `X-NAWA-Balance-Warning` | `low_balance` when below $5 |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Provider` | AI provider used (`allam`, `claude`, `gemini`) |
| `X-NAWA-Latency` | Server-side latency in milliseconds |
| `X-NAWA-Calibration-Version` | Calibration version applied (Enterprise only) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique classification ID (prefixed `cls_`) |
| `object` | string | Always `"classification"` |
| `intent` | string | `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language code (`ar`, `en`) |
| `dialect` | string or null | `gulf`, `egyptian`, `levantine`, `msa`. Null if not Arabic. |
| `dialect_confidence` | number or null | Confidence score (0-1). Null if not Arabic. |
| `requires_response` | boolean | Whether this comment needs a reply |
| `priority` | string | Priority level for response queue |
| `suggested_reply` | object | Contains `text` (reasoning/suggestion) and `direction` (`rtl` or `ltr`) |
| `provider` | string | AI provider used (`allam`, `claude`, `gemini`) |
| `model` | string | Specific model version used |
| `fallback_used` | boolean | Whether the fallback provider was used |
| `tokens_used` | number or null | Token count (when available) |
| `cost_usd` | number | Cost in USD |
| `credits_used` | number | Credits deducted |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, invalid JSON, invalid `provider` query param |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 405 | `method_not_allowed` | Using a method other than POST |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
