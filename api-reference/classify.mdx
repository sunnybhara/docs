Classify an Arabic comment with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply powered by HUMAIN's ALLaM model with automatic fallback to Claude.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`). Free keys are not charged credits.
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Override the AI provider: `claude`, `gemini`, `allam`. Useful for A/B testing. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The comment text to classify. Max 5,000 characters. |
| `context` | object | No | Additional context to improve classification accuracy. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer $NAWA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟",
    "context": {
      "platform": "youtube",
      "channel_id": "ch_123"
    }
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'متى الجزء الثاني؟',
  context: {
    platform: 'youtube',
    channelId: 'ch_123'
  }
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(
    text="متى الجزء الثاني؟",
    context={
        "platform": "youtube",
        "channel_id": "ch_123"
    }
)
```

</CodeGroup>

```json
    "fallback_used": false,
    "cost_usd": 0.006,
    "credits_used": 6
  },
  "success": true,
  "errors": [],
  "request_id": "req_nw_abc123def456"

```

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Balance` | Current credit balance in USD |
| `X-NAWA-Balance-Warning` | `low_balance` when below $5 |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost) |
| `X-NAWA-Provider` | Which AI provider handled the request |
| `X-NAWA-Environment` | `live` or `sandbox` |
| `X-NAWA-Calibration-Version` | Version of calibration thresholds used |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique classification ID (format: `cls_nw_xxx`) |
| `object` | string | Always `"classification"` |
| `intent` | string[] | Detected intents: `question`, `complaint`, `praise`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language code (e.g., `ar`, `en`) |
| `dialect` | string | `gulf`, `egyptian`, `levantine`, `maghrebi`, `msa` |
| `dialect_confidence` | number | Confidence score (0-1) |
| `requires_response` | boolean | Whether this comment needs a reply |
| `priority` | string | `low`, `medium`, `high` |
| `suggested_reply` | object or null | AI-generated reply with `text` and `direction` fields |
| `provider` | string | AI provider used: `allam`, `claude`, `gemini` |
| `model` | string | Model version used for classification |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `cost_usd` | number | Cost in USD (0 for free keys and cache hits) |
| `credits_used` | number | Credits deducted (0 for free keys and cache hits) |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, text too long, invalid JSON |
| 401 | `authentication_error` | Invalid, missing, expired, or revoked API key |
| 402 | `insufficient_credits` | No credits remaining (grace period exceeded) |
| 429 | `rate_limit_error` | Rate limit exceeded or sandbox cap reached |
| 500 | `api_error` | Internal or provider error |
