Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction. Supports semantic caching and multi-provider routing.

## Request

<Note>
  Cost: **$0.002** per request. Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider: `claude`, `gemini`, or `allam`. Useful for A/B testing. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Min 1 character. |
| `context` | object | No | Optional context object to improve classification accuracy. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST "https://api.trynawa.com/v1/classify?provider=allam" \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "ماشاء الله الفيديو حلو مرة"
  }'
```

```typescript TypeScript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({ apiKey: process.env.NAWA_API_KEY })

const { data, error } = await nawa.classify({
  text: 'ماشاء الله الفيديو حلو مرة'
})
```

```python Python
from nawa import Nawa

nawa = Nawa(api_key="your_api_key")

result = nawa.classify(
    text="ماشاء الله الفيديو حلو مرة"
)
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier (`req_nw_xxx` format) |
| `X-RateLimit-Limit` | Rate limit ceiling for current window |
| `X-RateLimit-Remaining` | Requests remaining in current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |
| `X-NAWA-Provider` | AI provider used: `allam`, `claude`, or `gemini` |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost) |
| `X-NAWA-Balance` | Current credit balance in USD |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_nw_xxx` format) |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents: `praise`, `question`, `complaint`, `suggestion`, `spam`, `other` |
| `sentiment` | string | `positive`, `negative`, `neutral`, `mixed` |
| `language` | string | Detected language: `ar`, `en`, `mixed` |
| `dialect` | string | Arabic dialect: `gulf`, `egyptian`, `levantine`, `maghrebi`, `msa`. Null if not Arabic. |
| `dialect_confidence` | number | Confidence score (0-1). Null if not Arabic. |
| `requires_response` | boolean | Whether the comment needs a reply |
| `priority` | string | `high`, `medium`, `low` |
| `suggested_reply` | object | Contains `text` (reply direction) and `direction` (`ltr` or `rtl`) |
| `provider` | string | AI provider used: `claude`, `gemini`, or `allam` |
| `model` | string | Model version used for classification |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | integer | Token count. Null if not applicable. |
| `cost` | number | Cost in USD |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text`, text too long |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `insufficient_credits` | No credits remaining |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
