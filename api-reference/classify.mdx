Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction. Supports semantic caching and multi-provider routing.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, or `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Must be non-empty. |
| `context` | object | No | Optional context object for classification. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/classify \
  -H "Authorization: Bearer nawa_test_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "متى الجزء الثاني؟"
  }'
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/classify', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer nawa_test_sk_xxx',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'متى الجزء الثاني؟'
  })
})

const data = await response.json()
console.log(data.result.intent)    // ["question"]
console.log(data.result.dialect)   // "gulf"
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/classify",
    headers={
        "Authorization": "Bearer nawa_test_sk_xxx",
        "Content-Type": "application/json"
    },
    json={
        "text": "متى الجزء الثاني؟"
    }
)

data = response.json()
print(data["result"]["intent"])    # ["question"]
print(data["result"]["dialect"])   # "gulf"
```

</CodeGroup>

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier (`req_nw_xxx` format) |
| `X-NAWA-Provider` | AI provider used: `claude`, `gemini`, or `allam` |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Latency` | Request processing time in milliseconds |
| `X-NAWA-Calibration-Version` | Calibration version applied, or `default` |
| `X-NAWA-Environment` | `sandbox` or `live` |
| `X-RateLimit-Limit` | Maximum requests allowed per minute for your tier |
| `X-RateLimit-Remaining` | Requests remaining in the current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID (`cls_nw_xxx` format) |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents (e.g., `["question"]`, `["question", "complaint"]`) |
| `sentiment` | string | `positive`, `negative`, `neutral`, or `mixed` |
| `language` | string | Detected language: `ar`, `en`, or `mixed` |
| `dialect` | string \| null | Arabic dialect: `gulf`, `egyptian`, `levantine`, `maghrebi`, or `msa`. Null for non-Arabic text. |
| `dialect_confidence` | number \| null | Confidence score (0-1). Null for non-Arabic text. |
| `requires_response` | boolean | Whether the comment warrants a reply |
| `priority` | string | `high`, `medium`, or `low` |
| `suggested_reply` | object | Contains `text` (suggested reply) and `direction` (`ltr` or `rtl`) |
| `provider` | string | AI provider used: `claude`, `gemini`, or `allam` |
| `model` | string | Specific model version used |
| `fallback_used` | boolean | Whether a fallback provider was used |
| `tokens_used` | integer \| null | Token count, if available |
| `cost_usd` | number | Cost in USD for this request |
| `credits_used` | integer | Credits deducted from your balance |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing or empty `text`, invalid `provider` value, malformed JSON |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `permission_error` | Insufficient credits |
| 405 | `invalid_request_error` | Non-POST request method |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
