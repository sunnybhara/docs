---

Classify Arabic or English text with a single request. Returns intent, sentiment, dialect, priority, and a suggested reply direction. Supports semantic caching and per-user calibration.

## Request

<Note>
  Cost: **$0.006** per request (6 credits). Semantic cache hits are free (`X-NAWA-Cache: HIT`).
</Note>

### Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | `Bearer nawa_live_sk_xxx` or `Bearer nawa_test_sk_xxx` |
| `Content-Type` | Yes | `application/json` |
| `X-Request-Id` | No | Client-supplied correlation ID |

### Query parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `provider` | string | No | Force a specific AI provider for A/B testing: `claude`, `gemini`, `allam`. |

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | Yes | The text to classify. Must be non-empty. |
| `context` | object | No | Optional context object for classification. |

### Example request

<CodeGroup>

```bash cURL
curl -X POST "https://api.trynawa.com/v1/classify" \
  -H "Authorization: Bearer nawa_live_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "ماشاء الله الفيديو حلو مرة"
  }'
```

```typescript TypeScript
const response = await fetch("https://api.trynawa.com/v1/classify", {
  method: "POST",
  headers: {
    Authorization: "Bearer nawa_live_sk_xxx",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    text: "ماشاء الله الفيديو حلو مرة",
  }),
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/classify",
    headers={"Authorization": "Bearer nawa_live_sk_xxx"},
    json={"text": "ماشاء الله الفيديو حلو مرة"},

data = response.json()
```

### Response headers

| Header | Description |
|--------|-------------|
| `X-Request-Id` | Unique request identifier for debugging |
| `X-NAWA-Provider` | AI provider used: `claude`, `gemini`, or `allam` |
| `X-NAWA-Cache` | `HIT` if served from semantic cache (no cost), `MISS` otherwise |
| `X-NAWA-Latency` | Processing time in milliseconds |
| `X-NAWA-Calibration-Version` | Per-user calibration version applied |
| `X-RateLimit-Limit` | Rate limit ceiling for the current window |
| `X-RateLimit-Remaining` | Requests remaining in the current window |
| `X-RateLimit-Reset` | Window reset time (RFC 3339) |

### Result fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Classification ID in `cls_nw_xxx` format |
| `object` | string | Always `classification` |
| `intent` | string[] | Detected intents, e.g. `["praise"]` or `["question", "complaint"]` |
| `sentiment` | string | `positive`, `negative`, `neutral`, or `mixed` |
| `language` | string | Detected language: `ar`, `en`, or `mixed` |
| `dialect` | string or null | Arabic dialect: `gulf`, `egyptian`, `levantine`, `maghrebi`, or `msa`. Null if not Arabic. |
| `dialect_confidence` | number or null | Confidence score (0-1). Null if not Arabic. |
| `requires_response` | boolean | Whether the comment warrants a reply |
| `priority` | string | `high`, `medium`, or `low` |
| `suggested_reply` | object | Contains `text` (reply direction) and `direction` (`ltr` or `rtl`) |
| `provider` | string | AI provider used: `claude`, `gemini`, or `allam` |
| `model` | string | Specific model version |
| `fallback_used` | boolean | Whether a fallback provider was needed |
| `tokens_used` | integer or null | Token count, if available |
| `cost_usd` | number | Cost in USD |
| `credits_used` | integer | Credits consumed |

### Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Missing `text` or invalid request body |
| 401 | `authentication_error` | Invalid or missing API key |
| 402 | `permission_error` | Insufficient credits |
| 429 | `rate_limit_error` | Rate limit exceeded |
| 500 | `api_error` | Internal or provider error |
