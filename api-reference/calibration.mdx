Manage calibration thresholds that fine-tune how NAWA classifies your content. Calibration lets you adjust confidence thresholds for sentiment, intent, and dialect detection, set preferred AI providers, and define custom labels.

<Note>
  Calibration is available on **Enterprise** and **Enterprise+** tiers only. Other tiers receive a `403` error. This endpoint is free (0 credits).
</Note>

## Retrieve calibration

<CodeGroup>

```bash cURL
curl https://api.trynawa.com/v1/calibration \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/calibration', {
  headers: { 'Authorization': `Bearer ${process.env.NAWA_API_KEY}` }
})
const data = await response.json()
```

```python Python
import requests

response = requests.get(
    "https://api.trynawa.com/v1/calibration",
    headers={"Authorization": "Bearer your_api_key"}
)
data = response.json()
```

</CodeGroup>

### Response (200)

```json
{
  "success": true,
  "result": {
    "calibration": {
      "sentiment_threshold": 0.7,
      "intent_threshold": 0.6,
      "dialect_threshold": 0.5
    },
    "defaults": {
      "sentiment_threshold": 0.7,
      "intent_threshold": 0.6,
      "dialect_threshold": 0.5
    },
    "history": []
  },
  "errors": [],
  "request_id": "req_nw_cal_abc123"
}
```

## Update calibration

Send a partial update with only the fields you want to change.

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `sentiment_threshold` | number | No | Confidence threshold for sentiment (0.0-1.0). Default: 0.7. |
| `intent_threshold` | number | No | Confidence threshold for intent (0.0-1.0). Default: 0.6. |
| `dialect_threshold` | number | No | Confidence threshold for dialect (0.0-1.0). Default: 0.5. |
| `preferred_provider` | string | No | Preferred AI provider: `claude`, `gemini`, `allam`. |
| `custom_labels` | object | No | Custom label mappings (max 20 entries). Keys and values must be strings. |

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/calibration \
  -H "Authorization: Bearer nawa_live_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "sentiment_threshold": 0.8,
    "preferred_provider": "allam"
  }'
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/calibration', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.NAWA_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    sentiment_threshold: 0.8,
    preferred_provider: 'allam'
  })
})
const data = await response.json()
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/calibration",
    headers={"Authorization": "Bearer your_api_key"},
    json={
        "sentiment_threshold": 0.8,
        "preferred_provider": "allam"
    }
)
data = response.json()
```

</CodeGroup>

### Response (200)

```json
{
  "success": true,
  "result": {
    "calibration": {
      "sentiment_threshold": 0.8,
      "intent_threshold": 0.6,
      "dialect_threshold": 0.5,
      "preferred_provider": "allam"
    }
  },
  "errors": [],
  "request_id": "req_nw_cal_def456"
}
```

## Reset calibration

Reset all calibration settings to their defaults.

<CodeGroup>

```bash cURL
curl -X DELETE https://api.trynawa.com/v1/calibration \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/calibration', {
  method: 'DELETE',
  headers: { 'Authorization': `Bearer ${process.env.NAWA_API_KEY}` }
})
const data = await response.json()
```

```python Python
import requests

response = requests.delete(
    "https://api.trynawa.com/v1/calibration",
    headers={"Authorization": "Bearer your_api_key"}
)
data = response.json()
```

</CodeGroup>

### Response (200)

```json
{
  "success": true,
  "result": {
    "reset": true
  },
  "errors": [],
  "request_id": "req_nw_cal_ghi789"
}
```

## Error responses

| Status | Type | When |
|--------|------|------|
| 400 | `invalid_request_error` | Invalid threshold value, unknown fields, or invalid provider |
| 401 | `authentication_error` | Invalid or missing API key |
| 403 | `permission_error` | Tier is below Enterprise |
| 429 | `rate_limit_error` | Rate limit exceeded |
