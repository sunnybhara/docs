Configure custom calibration thresholds to fine-tune classification confidence scores for your use case. Available to **Enterprise** and **Enterprise+** tiers only.

Calibration thresholds let you adjust the minimum confidence required for sentiment, intent, and dialect classifications. NAWA applies your thresholds automatically to every `/v1/classify` response, including cache hits.

## Get calibration

<Note>
  Requires Enterprise or Enterprise+ tier.
</Note>

### Request

```bash cURL
curl https://api.trynawa.com/v1/calibration \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

### Response (200)

```json
{
  "success": true,
  "result": {
    "calibration": {
      "sentiment_threshold": 0.8,
      "intent_threshold": 0.7,
      "dialect_threshold": 0.6
    },
    "defaults": {
      "sentiment_threshold": 0.7,
      "intent_threshold": 0.6,
      "dialect_threshold": 0.5
    },
    "history": [
      {
        "changed_at": "2026-03-10T14:30:00Z",
        "changes": {"sentiment_threshold": {"from": 0.7, "to": 0.8}}
      }
    ]
  },
  "request_id": "req_nw_cal_abc12345678901234567"
}
```

## Update calibration

Partially update one or more thresholds. Only include the fields you want to change.

### Request

<CodeGroup>

```bash cURL
curl -X POST https://api.trynawa.com/v1/calibration \
  -H "Authorization: Bearer nawa_live_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "sentiment_threshold": 0.85,
    "intent_threshold": 0.7
  }'
```

```typescript TypeScript
const response = await fetch('https://api.trynawa.com/v1/calibration', {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.NAWA_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    sentiment_threshold: 0.85,
    intent_threshold: 0.7
  })
})
```

```python Python
import requests

response = requests.post(
    "https://api.trynawa.com/v1/calibration",
    json={"sentiment_threshold": 0.85, "intent_threshold": 0.7},
    headers={"Authorization": "Bearer your_api_key"}
)
```

</CodeGroup>

### Response (200)

```json
{
  "success": true,
  "result": {
    "calibration": {
      "sentiment_threshold": 0.85,
      "intent_threshold": 0.7,
      "dialect_threshold": 0.5
    }
  },
  "request_id": "req_nw_cal_def12345678901234567"
}
```

### Body parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `sentiment_threshold` | number | No | Minimum confidence for sentiment (0--1). Default: `0.7`. |
| `intent_threshold` | number | No | Minimum confidence for intent (0--1). Default: `0.6`. |
| `dialect_threshold` | number | No | Minimum confidence for dialect (0--1). Default: `0.5`. |

## Reset calibration

Reset all thresholds to system defaults.

```bash cURL
curl -X DELETE https://api.trynawa.com/v1/calibration \
  -H "Authorization: Bearer nawa_live_sk_xxx"
```

### Response (200)

```json
{
  "success": true,
  "result": {
    "reset": true
  },
  "request_id": "req_nw_cal_ghi12345678901234567"
}
```

## Error responses

| Status | Code | When |
|--------|------|------|
| 400 | `invalid_field` | Threshold value out of range (must be 0--1) |
| 401 | `authentication_error` | Invalid or missing API key |
| 403 | `tier_insufficient` | Account is not on Enterprise or Enterprise+ tier |
| 429 | `rate_limit_error` | Rate limit exceeded |
