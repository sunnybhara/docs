The official NAWA Python SDK with synchronous and asynchronous support, auto-pagination, and comprehensive type hints.

## Installation

<CodeGroup>

```bash pip
pip install nawa-sdk
```

```bash poetry
poetry add nawa-sdk
```

```bash uv
uv add nawa-sdk
```

</CodeGroup>

**Requirements:** Python 3.8+

## Quick start

```python
from nawa import Nawa

nawa = Nawa(api_key="nawa_live_sk_xxx")

# Classify an Arabic comment
result, error = nawa.classify(
    text="ما شاء الله عليك، محتوى رهيب!",
    platform="youtube"
)

if error:
    print(error.type, error.message)
else:
    print(result.intent)    # "praise"
    print(result.dialect)   # "gulf"
    print(result.sentiment) # "positive"
```

<Note>
  The SDK returns `(result, error)` tuples. Check `error` first before accessing `result`.
</Note>

## Async support

The SDK provides a fully async client powered by `httpx`:

```python
from nawa import AsyncNawa
import asyncio

async def main():
    nawa = AsyncNawa(api_key="nawa_live_sk_xxx")

    result, error = await nawa.classify(
        text="متى الجزء الثاني؟",
        platform="youtube"
    )

    if result:
        print(result.intent)   # "question"
        print(result.dialect)  # "gulf"

    await nawa.close()

asyncio.run(main())
```

<Tip>
  Use `AsyncNawa` with `async with` for automatic cleanup:

  ```python
  async with AsyncNawa(api_key="nawa_live_sk_xxx") as nawa:
      result, error = await nawa.classify(text="متى الجزء الثاني؟", platform="youtube")
  ```
</Tip>

## Translate text

Translate between English and Arabic with dialect and tone control:

```python
result, error = nawa.translate.create(
    text="Hello, how are you?",
    target="ar",
    dialect="gulf",
    tone="casual"
)

if result:
    print(result.translated_text)  # "هلا، كيف حالك؟"
    print(result.source_language)  # "en"
    print(result.target_dialect)   # "gulf"
```

All parameters:

```python
result, error = nawa.translate.create(
    text="Subscribe for more!",
    target="ar",
    source="en",            # Optional: "ar", "en", or "auto" (default)
    dialect="egyptian",      # Optional: "gulf", "egyptian", "levantine", "msa" (default)
    tone="social_media",     # Optional: "formal" (default), "casual", "social_media"
    platform="youtube",      # Optional: "youtube", "instagram", "twitter", "facebook"
    metadata={"key": "val"}  # Optional: arbitrary metadata
)
```

## Detect language and dialect

Lightweight detection with no AI model call (under 100ms):

```python
result, error = nawa.detect.create(text="وش رايكم بالمحتوى الجديد؟")

if result:
    print(result.language)           # "ar"
    print(result.dialect)            # "gulf"
    print(result.dialect_confidence) # 0.95
    print(result.script)            # "arabic"
    print(result.text_direction)    # "rtl"
    print(result.word_count)        # 4
```

## Moderate content

Dialect-aware content moderation with 7 categories and severity scores:

```python
result, error = nawa.moderate.create(
    text="هذا محتوى رائع جدا",
    strictness="medium"
)

if result:
    print(result.is_safe)       # True
    print(result.verdict)       # "safe"
    print(result.severity)      # "none"
```

With platform context:

```python
result, error = nawa.moderate.create(
    text="comment text here",
    strictness="high",           # "low", "medium" (default), "high"
    platform="youtube",          # Optional
    context="youtube comment"    # Optional: free-form context string
)

if result and not result.is_safe:
    print(result.flags)          # ["profanity", "harassment"]
    print(result.severity_score) # 0.87
    print(result.flagged_terms)  # ["term1", "term2"]
    print(result.cleaned_text)   # "text with *** replacing flagged terms"
```

## List comments with auto-pagination

```python
# Single page
result, error = nawa.comments.list(
    platform="youtube",
    intent="question",
    limit=25
)

# Auto-pagination (iterator)
for comment in nawa.comments.list(platform="youtube"):
    print(comment.text, comment.intent)

# Async auto-pagination
async for comment in nawa.comments.list(platform="youtube"):
    print(comment.text, comment.intent)
```

## Error handling

```python
from nawa import Nawa
from nawa.errors import NawaAuthError, NawaRateLimitError, NawaCreditsError

nawa = Nawa(api_key="your_api_key")
result, error = nawa.classify(text="...", platform="youtube")

if error:
    if isinstance(error, NawaAuthError):
        print(f"Auth error: {error.code}")
    elif isinstance(error, NawaRateLimitError):
        print(f"Retry after: {error.retry_after}s")
    elif isinstance(error, NawaCreditsError):
        print(f"Buy credits: {error.suggested_action}")
    else:
        print(f"API error: {error.message}")
```

## Configuration

```python
nawa = Nawa(
    api_key="nawa_live_sk_xxx",

    # Retry configuration
    max_retries=3,         # Default: 3
    retry_delay=1.0,       # Base delay in seconds. Default: 1.0
    retry_backoff="exponential",  # "exponential", "linear", "none"

    # Timeout
    timeout=30.0,          # Request timeout in seconds. Default: 30.0

    # Base URL override (for testing)
    base_url="https://api.trynawa.com",  # Default
)
```
