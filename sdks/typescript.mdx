The official NAWA TypeScript SDK provides a type-safe client with `{ data, error }` returns, auto-pagination, and built-in retry logic.

## Installation

<CodeGroup>

```bash npm
npm install @nawalabs/sdk
```

```bash pnpm
pnpm add @nawalabs/sdk
```

```bash yarn
yarn add @nawalabs/sdk
```

```bash bun
bun add @nawalabs/sdk
```

</CodeGroup>

## Quick start

```typescript
import { Nawa } from '@nawalabs/sdk'

const nawa = new Nawa({
  apiKey: process.env.NAWA_API_KEY  // nawa_live_sk_xxx or nawa_test_sk_xxx
})

// Classify an Arabic comment
const { data, error } = await nawa.classify({
  text: 'ما شاء الله عليك، محتوى رهيب!',
  platform: 'youtube'
})

if (error) {
  console.error(error.type, error.message)
} else {
  console.log(data.intent)    // "praise"
  console.log(data.dialect)   // "gulf"
  console.log(data.sentiment) // "positive"
}
```

<Note>
  The SDK uses `{ data, error }` returns instead of throwing exceptions. This makes error handling explicit and avoids uncaught promise rejections.
</Note>

## Translate text

Translate between English and Arabic with dialect and tone control:

```typescript
const { data, error } = await nawa.translate.create({
  text: 'Hello, how are you?',
  target: 'ar',
  dialect: 'gulf',
  tone: 'casual'
})

if (data) {
  console.log(data.translated_text)  // "هلا، كيف حالك؟"
  console.log(data.source_language)  // "en"
  console.log(data.target_dialect)   // "gulf"
}
```

All parameters:

```typescript
const { data, error } = await nawa.translate.create({
  text: 'Subscribe for more!',
  target: 'ar',
  source: 'en',           // Optional: "ar", "en", or "auto" (default)
  dialect: 'egyptian',     // Optional: "gulf", "egyptian", "levantine", "msa" (default)
  tone: 'social_media',   // Optional: "formal" (default), "casual", "social_media"
  platform: 'youtube',    // Optional: "youtube", "instagram", "twitter", "facebook"
  metadata: { key: 'val' } // Optional: arbitrary metadata
})
```

## Detect language and dialect

Lightweight detection with no AI model call (under 100ms):

```typescript
const { data, error } = await nawa.detect.create({
  text: 'وش رايكم بالمحتوى الجديد؟'
})

if (data) {
  console.log(data.language)           // "ar"
  console.log(data.dialect)            // "gulf"
  console.log(data.dialect_confidence) // 0.95
  console.log(data.script)            // "arabic"
  console.log(data.text_direction)    // "rtl"
  console.log(data.word_count)        // 4
}
```

## Moderate content

Dialect-aware content moderation with 7 categories and severity scores:

```typescript
const { data, error } = await nawa.moderate.create({
  text: 'هذا محتوى رائع جدا',
  strictness: 'medium'
})

if (data) {
  console.log(data.is_safe)       // true
  console.log(data.verdict)       // "safe"
  console.log(data.severity)      // "none"
  console.log(data.categories)    // { profanity: false, hate_speech: false, ... }
}
```

With platform context:

```typescript
const { data, error } = await nawa.moderate.create({
  text: 'comment text here',
  strictness: 'high',           // "low", "medium" (default), "high"
  platform: 'youtube',          // Optional
  context: 'youtube comment'    // Optional: free-form context string
})

if (data && !data.is_safe) {
  console.log(data.flags)          // ["profanity", "harassment"]
  console.log(data.severity_score) // 0.87
  console.log(data.flagged_terms)  // ["term1", "term2"]
  console.log(data.cleaned_text)   // "text with *** replacing flagged terms"
}
```

## List comments with auto-pagination

```typescript
// Single page
const { data, error } = await nawa.comments.list({
  platform: 'youtube',
  intent: 'question',
  limit: 25
})

// Auto-pagination (async iterator)
for await (const comment of nawa.comments.list({ platform: 'youtube' })) {
  console.log(comment.text, comment.intent)
}
```

## Error handling

The SDK provides typed errors for each error category:

```typescript
import { Nawa, NawaAuthError, NawaRateLimitError, NawaCreditsError } from '@nawalabs/sdk'

const { data, error } = await nawa.classify({ text: '...', platform: 'youtube' })

if (error) {
  if (error instanceof NawaAuthError) {
    // Invalid, expired, or revoked API key
    console.error('Auth error:', error.code) // "invalid_api_key"
  } else if (error instanceof NawaRateLimitError) {
    // Rate limit exceeded - use error.retryAfter
    console.log(`Retry after ${error.retryAfter}ms`)
  } else if (error instanceof NawaCreditsError) {
    // Insufficient credits
    console.error('Buy credits:', error.suggestedAction)
  } else {
    // Server error - safe to retry
    console.error('API error:', error.message)
  }
}
```

## Configuration

```typescript
const nawa = new Nawa({
  apiKey: process.env.NAWA_API_KEY,

  // Retry configuration
  maxRetries: 3,          // Default: 3
  retryDelay: 1000,       // Base delay in ms. Default: 1000
  retryBackoff: 'exponential', // "exponential" | "linear" | "none"

  // Timeout
  timeout: 30_000,        // Request timeout in ms. Default: 30000

  // Base URL override (for testing)
  baseUrl: 'https://api.trynawa.com' // Default
})
```
