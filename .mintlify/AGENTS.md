# NAWA Documentation Agent Instructions

## Product

- NAWA is an Arabic-first AI comment classification API
- Supported dialects: Gulf, Egyptian, Levantine, MSA
- API base URL: api.trynawa.com/v1
- Key prefixes: nawa_test_sk_ (free), nawa_live_sk_ (live)

## Style

- Never use em dashes in any content
- Use "free" and "live" for key tiers, never "sandbox" or "production"
- Use "NAWA" not "Nawa" or "nawa" when referring to the product name in prose
- Tone: direct, developer-focused, concise
- Do not reference internal metrics like gross margin

## Code examples

- Always include code examples in curl, Python, and TypeScript

## Pricing

- When referencing pricing, use credits model ($0.005/comment for classify)
